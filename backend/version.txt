## 二、功能说明

### 2.1 Start2 版本（types/process_start.py）

Start2是带LLM调用的start阶段处理函数，新增功能：

| 功能 | 说明 |
|------|------|
| LLM调用 | 安全检查通过后，调用LLM分析用户意图 |
| 状态图标 | 根据条件返回不同的状态图标（✅/❓/⚠️/🚫） |
| 二次yield | 第一次yield start_data，第二次yield content + 状态 |
| 异常处理 | LLM调用失败时有fallback逻辑 |

### 2.2 状态图标判断逻辑

```
优先级（从高到低）：
1. blocked=true → 🚫 （安全拦截）
2. is_clear=false → ❓ （输入含混不清）
3. is_need_confirm=true → ⚠️ （有风险需确认）
4. 其他 → ✅ （正常，可进入thought）
```
