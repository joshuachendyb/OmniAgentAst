```typescript
interface Session {
  session_id: string;        // 会话唯一ID
  title: string;             // 会话标题

  title_source: "user" | "auto";  // ⭐ 标题来源（用户手动/自动生成）
  title_updated_at: string | null; // ⭐ 标题最后更新时间
```

3. **标题来源**：标记标题是用户手动设置还是自动生成

```typescript
// 独立的重试函数
const retryWithDelay = async (
  fn: () => Promise<void>,
  maxRetries: number,
  delay: number
) => {
  for (let i = 0; i < maxRetries; i++) {
    try {
      await fn();
      return;
    } catch (error) {
      if (i === maxRetries - 1) {
        throw error; // 最后一次失败，抛出错误
      }
      await new Promise(resolve => setTimeout(resolve, delay));
    }
  }
};
```

```
│  用户编辑标题                                                │
│     ↓                                                        │
│  输入框 onChange → setTitleInput(value)                    │
│     ↓                                                        │
│  onBlur 或 onInputSubmit → ensureTitlePersisted()         │
│     ↓                                                        │
│  检查1：标题未变化 → return                                 │
│  检查2：正在保存 → return                                   │
│     ↓                                                        │
│  防抖保存（debounce 1秒）                                    │
│     ↓                                                        │
│  sessionApi.updateSession(title, version)                  │
│     ↓                                                        │
│  ├─ 成功 → setSessionTitle(), setLastSavedTitle()         │
│  ├─ 409冲突 → 重新获取最新数据 + message提示               │
│  └─ 其他错误 → 检查重试次数 < 3 → 1秒后重试                │
└─────────────────────────────────────────────────────────────┘
```
