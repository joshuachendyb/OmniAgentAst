## 一、文档说明

### 1.1 目标

按照代码10大原则（SRP、DRY、KISS、SLAP、YAGNI + OCP、LSP、ISP、复用优先），对4个大文件进行拆分设计。

### 1.2 拆分原则

| 原则 | 要求 |
|------|------|
| **SRP** | 每个文件/类只做一件事 |
| **复制优先** | 拆分时复制函数代码，不是重写函数 |
| **依赖清晰** | 拆分后依赖关系明确，无循环依赖 |
| **可回滚** | 每个拆分步骤可独立回滚 |
| **原文件名不变** | 拆分后原文件名保持不变，只精简内容 |
| **函数命名准确** | 拆出来的函数命名必须名副其实 |
| **目录名称规范** | 拆出来的函数代码以主代码名称作为目录名称 |

### 1.3 拆分范围

| 文件 | 当前行数 | 目标行数 | 拆分数量 |
|------|---------|---------|---------|
| react_sse_wrapper.py | 629行 | ~200行 | 6个文件 |
| base_react.py | 499行 | ~200行 | 4个文件 |
| llm_core.py | 436行 | ~180行 | 4个文件 |
| react_handler_mixin.py | 372行 | ~50行 | 3个文件 |

```python
class BaseAgent(ReActHandlerMixin, ABC):
    def __init__(self, ...):
        # 委托给AgentInitializer
        AgentInitializer.initialize(self, ...)
        # 初始化ToolManager
        self._tool_manager = ToolManager(self)
        self._tool_manager.init_tools()
        # 初始化StepEmitter
        self._step_emitter = StepEmitter(self)
    
    async def run_stream(self, ...): ...  # 核心循环
    
    # 抽象方法
    @abstractmethod
    def _get_system_prompt(self): ...
    @abstractmethod
    def _get_task_prompt(self, task, context): ...
    
    # Hook方法
    def _on_session_init(self, task, context): ...
    def _on_before_loop(self, sys_prompt, task_prompt, context): ...
    def _on_after_loop(self): ...
    
    # 透传方法（保持向后兼容）
    def _emit_step(self, step):
        return self._step_emitter.emit(step)
    def _exit_with_error(self, step_count, error_type, error_message, recoverable=False):
        return self._step_emitter.exit_with_error(step_count, error_type, error_message, recoverable)
    def _check_interrupt(self, step_count, running_tasks):
        return self._step_emitter.check_interrupt(step_count, running_tasks)
    def _complete_tracked_task(self, success):
        return self._step_emitter.complete_task(success)
    def record_operation(self, operation_type, **kwargs):
        return self._step_emitter.record_operation(operation_type, **kwargs)
```
