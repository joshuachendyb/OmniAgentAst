## 三、后端及API部分

### 3.1 流式API接口

#### 3.1.1 流式对话接口
**功能点编号**: API-001  
**接口地址**: `http://localhost:8000/api/v1/chat/stream`  
**请求方法**: POST  
**请求参数**:
```typescript
{
  messages: [
    {
      role: 'user',
      content: string // 用户输入的消息内容
    }
  ],
  stream: boolean // 是否使用流式输出，固定为true
  task_id?: string // 任务ID，用于中断功能
}
```

**响应格式**: Server-Sent Events (SSE)
```
data: {"type":"thought","content":"正在思考...","timestamp":1713348823456}
data: {"type":"action","tool":"search","params":{"query":"OmniAgentAs"},"timestamp":1713348823567}
data: {"type":"observation","result":"找到了项目相关信息","timestamp":1713348823678}
data: {"type":"chunk","content":"Hello","timestamp":1713348823789}
data: {"type":"final","content":"完整回复内容","model":"gpt-4","timestamp":1713348823890}
```

**响应字段说明**:
| 字段 | 类型 | 说明 |
|------|------|------|
| type | string | 消息类型：thought/action/observation/chunk/final/error/interrupted |
| content | string | 消息内容 |
| tool | string | 工具名称（action类型） |
| params | Record<string, any> | 工具参数（action类型） |
| result | any | 执行结果（observation类型） |
| model | string | 使用的模型名称（final类型） |
| timestamp | number | 时间戳 |

#### 3.1.2 任务中断接口
