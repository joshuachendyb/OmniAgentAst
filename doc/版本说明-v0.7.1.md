### 3.2 设计目标

**当前代码的问题**：
- 意图检测和安全检查都硬编码在文件操作里，和文件操作绑死了，无法扩展
- 框架不够通用，如果只支持 file，框架就和文件助手绑死了

**OmniAgent 的定位**：
- OmniAgent 不是"文件操作助手"，而是一个**通用的多意图处理 Agent**
- 通过意图分类来决定处理路径，当前只实现了"文件操作"这一个意图，但框架层面支持所有意图类型的扩展

**核心原则**：
- 框架通用，实现专用
- 意图分类驱动
- 预留扩展接口
- 当前只实现 file 意图，其他类型预留

```python
    r"D:\OmniAgentAs-desk\backend\app\services\agent",
    r"D:\OmniAgentAs-desk\backend\app\api\v1",

    r"D:\OmniAgentAs-desk\backend\tests",
    r"D:\OmniAgentAs-desk\backend\tools",
]

def replace_imports(content):
    content = content.replace("app.services.file_operations.", "app.services.agent.")
    content = content.replace("app.services.file_operations", "app.services.agent")
    return content

def process_file(filepath):
    if not filepath.endswith('.py'):
        return False
    with open(filepath, 'r', encoding='utf-8') as f:
        original = f.read()
    replaced = replace_imports(original)
    if original != replaced:
        with open(filepath, 'w', encoding='utf-8') as f:
            f.write(replaced)
        return True
    return False
```
