### 2.2 问题P0-002: `getSessionTitle()` 函数缺失

**来源**: Git diff 第361-444行  
**背景说明**:
- 统一管理会话标题的来源优先级
- 优先级：API返回的title &gt; 用户修改过的标题 &gt; 消息内容 &gt; 默认标题
- 这是标题管理的核心逻辑函数

**为什么要增加**:
1. **后端API变更适配**：后端 `GET /api/v1/sessions/{session_id}/messages` 新增了 `title` 字段
2. **统一逻辑**：避免在多个地方重复写标题获取逻辑
3. **优先级管理**：明确规定不同来源的标题优先级

**函数功能**:
```typescript
const getSessionTitle = (
  sessionData: { title?: string; messages?: any[] },
  defaultTitle: string = '会话'
): string => {
  // 1. 最高优先级：API返回的title字段
  if (sessionData.title && sessionData.title.trim()) {
    return sessionData.title.trim();
  }
  
  // 2. 次优先级：消息内容（只在没有API title时使用）
  if (sessionData.messages && sessionData.messages.length &gt; 0) {
    const firstMessage = sessionData.messages[0];
    if (firstMessage?.content) {
      const contentTitle = firstMessage.content.substring(0, 30).trim();
      if (contentTitle) {
        return contentTitle;
      }
    }
  }
  
  // 3. 默认标题
  return defaultTitle;
};
```

**严重程度**: 🔴 P0-紧急  
**影响**: 无法正确获取和显示会话标题

### 2.3 问题P0-003: `generateNewSessionTitle()` 函数缺失

**来源**: Git diff 第446-466行  
**背景说明**:
- 智能生成有意义的会话标题，而不是简单的"新会话"
- 根据时间段生成不同的标题，提升用户体验

**为什么要增加**:
1. **UX优化**：智能标题比"新会话"更有意义，用户更容易识别
2. **时间细分**：清晨、上午、午间、下午、晚间、深夜，更精确
3. **带时间戳**：包含具体时间，方便排序和查找

**函数功能**:
```typescript
const generateNewSessionTitle = (): string =&gt; {
  const now = new Date();
  const hours = now.getHours();
  let timeOfDay = '';
  
  // 更精确的时间段划分
  if (hours &gt;= 5 &amp;&amp; hours &lt; 8) timeOfDay = '清晨';
  else if (hours &gt;= 8 &amp;&amp; hours &lt; 12) timeOfDay = '上午';
  else if (hours &gt;= 12 &amp;&amp; hours &lt; 14) timeOfDay = '午间';
  else if (hours &gt;= 14 &amp;&amp; hours &lt; 18) timeOfDay = '下午';
  else if (hours &gt;= 18 &amp;&amp; hours &lt; 21) timeOfDay = '晚间';
  else if (hours &gt;= 21 &amp;&amp; hours &lt; 24) timeOfDay = '深夜';
  else timeOfDay = '深夜'; // 0-5点
  
  const dateStr = `${now.getMonth() + 1}月${now.getDate()}日`;
  return `${dateStr} ${timeOfDay}会话 ${hours}:${now.getMinutes().toString().padStart(2, '0')}`;
};
```

**严重程度**: 🔴 P0-紧急  
**影响**: 新会话标题无法智能生成，用户体验差
