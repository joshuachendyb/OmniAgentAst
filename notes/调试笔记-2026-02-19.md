## 1 前端服务启动问题 【端口3000，Python静态服务器】2026-02-19 23:10:00

### 1.1 问题描述

启动前端服务时遇到多个问题：

**问题1**: `npm run dev` 启动失败，端口3000无法连接
- 现象：Vite dev server 在后台启动后没有正确绑定端口
- 原因：Vite preview 模式网络绑定问题

**问题2**: PowerShell 脚本执行错误
- 现象：多次出现 `$()` 变量引用错误
- 原因：PowerShell 转义问题

**问题3**: 端口3000被占用
- 现象：启动时提示端口占用
- 原因：之前的进程未完全退出

### 1.2 解决方案

最终使用 Python 静态服务器成功启动：

```bash
cd frontend
npm run build
cd dist
python -m http.server 3000
```

### 1.3 验证结果

✅ 前端服务: http://localhost:3000  
✅ 返回HTML页面正常  
✅ 端口监听状态：`TCP 0.0.0.0:3000 LISTENING`
