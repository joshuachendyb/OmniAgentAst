## 2 发现的问题清单 【问题分析】2026-03-01 00:44:31

### 2.1 问题 1：错误消息没有 isError 标记

**严重程度**: P1-高  
**位置**: `NewChatContainer.tsx:254-267`  
**类型**: 前端逻辑缺陷

**问题描述**:
- `onError` 回调没有设置 `isError: true` 标记
- 错误消息和正常消息样式完全一样
- 用户无法从视觉上区分错误消息

**当前代码**:
```typescript
// onError - 流式错误
useCallback((error: string) => {
  console.error("SSE 流式错误:", error);
  message.error({
    content: `AI 响应失败：${error}`,
    duration: 5,
  });

  setMessages((prev) => {
    const lastMessage = prev[prev.length - 1];
    if (lastMessage && lastMessage.role === "assistant") {
      const updated = [...prev];
      updated[updated.length - 1] = {
        ...lastMessage,
        content: lastMessage.content || "抱歉，发生了错误",
        isStreaming: false,
        // ❌ 缺少：isError: true
      };
      return updated;
    }
    return prev;
  });
  setLoading(false);
}, [])
```

**影响**:
- 用户不知道哪条消息出错了
- 错误消息没有视觉警示
- 用户体验差
