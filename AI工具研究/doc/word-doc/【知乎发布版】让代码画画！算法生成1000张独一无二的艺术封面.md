## 🧠 核心原理：代理系统（Agent System）

这个算法背后的核心思想叫做**"代理系统"**（Multi-Agent System）。

### 什么是代理？

想象画布上有80个小机器人（代理），每个机器人：
- 有自己的位置（x, y坐标）
- 有自己的运动方向（角度）
- 有自己的"画笔"（颜色、粗细）
- 有自己的"生命期"

它们按照简单的规则运动，留下轨迹，最终形成复杂的图案。

### 简单规则，复杂涌现

每个代理只遵循3个规则：

```javascript
// 规则1：根据噪声场决定方向
const angle = noise(x * noiseScale, y * noiseScale) * TWO_PI * 2;

// 规则2：按照这个方向移动
this.x += cos(angle) * speed;
this.y += sin(angle) * speed;

// 规则3：如果生命期结束或出界，就停止
if (this.life <= 0 || this.isOutOfBounds()) {
    this.dead = true;
}
```

**神奇之处**：单个代理的行为很简单，但当80个代理同时运行时，就会产生**涌现**（Emergence）—— 简单的规则产生复杂的、不可预测的美。

这就是生成艺术的本质：**控制与随机的平衡**。

## 💻 核心代码解析

### Step 1：初始化画布和代理

```javascript
function setup() {
    createCanvas(800, 600);
    background(250);
    
    // 设置随机种子，保证可复现
    seed = 42; // 每个数字对应一张独特的画
    randomSeed(seed);
    noiseSeed(seed);
    
    // 创建80个画笔代理
    agents = [];
    for (let i = 0; i < 80; i++) {
        agents.push(new DrawingAgent(random(width), random(height)));
    }
}
```

**关键点**：
- 使用 `randomSeed()` 和 `noiseSeed()` 确保**可复现性**
- 相同的种子永远生成相同的图
- 换种子 = 换一张新作品

### Step 2：代理类定义

```javascript
class DrawingAgent {
    constructor(x, y) {
        this.x = x;
        this.y = y;
        this.angle = random(TWO_PI);
        this.life = 300; // 生命期（步数）
        this.weight = random(0.5, 3); // 线条粗细
        
        // 颜色：石墨灰 + 蓝色点缀
        this.colors = [
            color(100, 100, 100, 150), // 灰
            color(70, 130, 180, 100),  // 蓝
            color(50, 50, 50, 200)     // 深灰
        ];
        this.color = random(this.colors);
    }
    
    update() {
        if (this.life <= 0) return;
        
        // 使用Perlin噪声决定运动方向
        // 噪声函数：输入相同的坐标，永远输出相同的值
        // 但相邻坐标的值平滑过渡，形成连续流动
        const n = noise(this.x * 0.005, this.y * 0.005);
        this.angle = n * TWO_PI * 4;
        
        // 记录当前位置
        const prevX = this.x;
        const prevY = this.y;
        
        // 移动
        this.x += cos(this.angle) * 2;
        this.y += sin(this.angle) * 2;
        
        // 绘制轨迹
        stroke(this.color);
        strokeWeight(this.weight * (this.life / 300)); // 越到后面越细
        line(prevX, prevY, this.x, this.y);
        
        this.life--;
    }
}
```

### Step 3：动画循环

```javascript
function draw() {
    // 更新所有代理
    let allDead = true;
    for (let agent of agents) {
        if (!agent.dead) {
            agent.update();
            allDead = false;
        }
    }
    
    // 所有代理都停止后，结束动画
    if (allDead) {
        noLoop();
        console.log("创作完成！");
    }
}
```

**完整代码约100行**，就能生成这种有机线条艺术！

## 🎯 为什么用算法生成艺术？

### 1. 独一无二

传统设计：你用这个模板，别人也用，容易"撞图"

算法艺术：每个随机种子（seed）对应一张**世界上唯一的作品**

```javascript
// Seed 42 的作品
randomSeed(42);

// Seed 123 的作品（完全不同）
randomSeed(123);
```

### 2. 无限生成

想要100张封面？只需要改一个数字：

```javascript
for (let seed = 1; seed <= 100; seed++) {
    randomSeed(seed);
    // 生成并保存...
}
```

**批量生成，每张都不同**。

### 3. 参数化控制

通过调整参数，可以控制风格：

```javascript
let params = {
    agentCount: 80,      // 代理数量：多=密集，少=稀疏
    noiseScale: 0.005,   // 噪声尺度：大=波动快，小=平滑
    speed: 2,            // 运动速度：快=长线条，慢=细腻
    turbulence: 2        // 扰动强度：高=混乱，低=有序
};
```

调整这些参数，就能从"抽象 chaos"到"极简秩序"之间自由切换。

### 4. 数学之美

这种艺术背后有深厚的数学基础：
- **Perlin噪声**：自然纹理的数学模拟
- **涌现理论**：简单规则的复杂结果
- **混沌理论**：对初始条件敏感，不可预测但有序

每一张图都是**数学的诗**。

## 🎓 进阶：理解Perlin噪声

刚才的代码中最关键的一行是：

```javascript
let angle = noise(x * 0.005, y * 0.005) * TWO_PI * 4;
```

### 什么是Perlin噪声？

Perlin噪声是一种**平滑随机函数**。

**普通随机**：
- random(0, 1) → 0.3
- random(0, 1) → 0.9
- random(0, 1) → 0.1
（完全无序，跳跃性）

**Perlin噪声**：
- noise(0.1) → 0.5
- noise(0.2) → 0.52
- noise(0.3) → 0.55
（平滑过渡，连续性）

**形象比喻**：
- 普通随机像电视雪花
- Perlin噪声像地形高度图

### 噪声尺度（Noise Scale）

```javascript
noise(x * 0.001)  // 大范围波动 → 平滑
noise(x * 0.01)   // 中等范围 → 适中
noise(x * 0.1)    // 小范围波动 → 细节丰富
```

**越大越细节，越小越平滑**。

掌握了基础原理后，你可以创造无限可能：

### 1. 流场可视化（Flow Field）

```javascript
// 在背景上绘制噪声场的方向
for (let x = 0; x < width; x += 20) {
    for (let y = 0; y < height; y += 20) {
        let angle = noise(x * 0.01, y * 0.01) * TWO_PI;
        line(x, y, x + cos(angle) * 15, y + sin(angle) * 15);
    }
}
```

### 2. 粒子系统（Particle System）

给代理添加物理属性：
- 速度（velocity）
- 加速度（acceleration）
- 力场（force field）

### 3. 递归分形（Recursive Fractals）

```javascript
function branch(len) {
    line(0, 0, 0, -len);
    translate(0, -len);
    
    if (len > 10) {
        push();
        rotate(PI/6);
        branch(len * 0.7);
        pop();
        
        push();
        rotate(-PI/6);
        branch(len * 0.7);
        pop();
    }
}
```

生成树状分形结构。

### 4. 几何变换（Geometric Transformations）

- 旋转、缩放、平移
- 对称、镜像、重复
- 扭曲、变形、投影
