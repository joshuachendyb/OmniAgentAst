### 2.1 分支1：UniversalReactAgent（通用Agent）

```
UniversalReactAgent
  混入3个mixin:
    ├── ToolStepMixin      ← 编排一次工具执行
    ├── ReactAgentMixin    ← 初始化逻辑
    └── RollbackMixin      ← 回滚能力（分支3）
  实现4个抽象方法:
    ├── _get_llm_response()   → self._call_llm()
    ├── _execute_tool()       → execute_tool_with_unified_retry()
    ├── _get_system_prompt()  → self.prompts.get_system_prompt()
    └── _get_task_prompt()    → self.prompts.get_task_prompt()
  特点:
    - 配置驱动（AgentConfig）
    - 支持回滚（混入RollbackMixin）
    - 取代原来9个重复的Agent子类
```

### 2.2 分支2：DesktopReactAgent（桌面Agent）

```
DesktopReactAgent
  混入2个mixin（无Rollback）:
    ├── ToolStepMixin      ← 编排一次工具执行
    └── ReactAgentMixin    ← 初始化逻辑
  实现4个抽象方法:
    ├── _get_llm_response()   → self._call_llm()
    ├── _execute_tool()       → execute_tool_with_unified_retry()
    ├── _get_system_prompt()  → self.prompts.get_system_prompt()
    └── _get_task_prompt()    → self.prompts.get_task_prompt()
  特点:
    - 桌面操作专用
    - 不支持回滚（桌面操作不需要）
    - 硬编码 DesktopPrompts
```

### 2.3 分支3：RollbackMixin（回滚能力）

```
RollbackMixin
  提供方法:
    └── rollback(step_number) → execute_tool_with_unified_retry()
  被谁混入:
    ├── UniversalReactAgent  ✅ 混入（文件操作需要回滚）
    └── DesktopReactAgent    ❌ 不混入（桌面操作不需要）
  依赖:
    └── tool_executor.py::execute_tool_with_unified_retry()
```

**关键区别**：
- 分支1（UniversalReactAgent）混入了RollbackMixin → 支持回滚
- 分支2（DesktopReactAgent）不混入RollbackMixin → 不支持回滚
- 分支3（RollbackMixin）本身不是Agent，是能力插件，被分支1混入

BaseAgent (ABC) + ReActHandlerMixin          ← base_react.py

## 三、每个文件的职责

| 角色 | 文件 | 类 | 职责 | 关键代码 |
|------|------|-----|------|---------|
| **主干** | `base_react.py` | `BaseAgent` | ReAct循环骨架，定义抽象方法 | `run_stream()` + 4个抽象方法 |
| **分支1** | `universal_react.py` | `UniversalReactAgent` | 通用Agent，混入全部mixin，配置驱动 | `_execute_tool()` → `execute_tool_with_unified_retry()` |
| **分支2** | `desktop_react.py` | `DesktopReactAgent` | 桌面Agent，不混入Rollback | `_execute_tool()` → `execute_tool_with_unified_retry()` |
| **分支3** | `rollback_mixin.py` | `RollbackMixin` | 可插拔回滚能力，被分支1混入 | `rollback()` → `execute_tool_with_unified_retry()` |

## 四、调用流程

```
Agent.run(task)
    │
    ▼
BaseAgent.run_stream()          ← ReAct循环（定义在base_react.py）
    │
    │ 每轮循环:
    │   1. _get_llm_response()   ← 子类实现（调LLM）
    │   2. parse → action
    │   3. _execute_tool(action)  ← 子类实现（调工具）
    │       │
    │       └──→ tool_executor.py::execute_tool_with_unified_retry()
    │                 │
    │                 └──→ tool_retry_engine.py::execute_tool_with_retry()
    │
    │ 观察结果 → 继续循环/退出
    │
    ▼
AgentResult
```

## 六、Agent层 vs Tool层

六、Agent层 vs Tool层

```
┌─────────────────────────────────────────┐
│          Agent层（这4个文件在这里）        │
│                                         │
│  base_react.py        ← ReAct循环骨架   │
│  universal_react.py   ← 通用Agent实现    │
│  desktop_react.py     ← 桌面Agent实现    │
│  rollback_mixin.py    ← Agent回滚能力    │
│                                         │
│  职责：思考→决策→调工具→观察→循环        │
└──────────────┬──────────────────────────┘
               │ _execute_tool()
               ▼
┌─────────────────────────────────────────┐
│          Tool层（另外的文件）              │
│                                         │
│  tool_executor.py     ← 工具执行统一入口 │
│  tool_retry_engine.py ← 工具重试引擎     │
│  各category_tools.py  ← 工具实现         │
│                                         │
│  职责：接收action+params → 执行 → 返回结果│
└─────────────────────────────────────────┘
```

```
UniversalReactAgent   DesktopReactAgent   RollbackMixin
       │                    │                  │
       │ 继承               │ 继承              │ 混入
       │                    │                  ↓
       │                    │          (混入UniversalReactAgent)
       └──────────┬─────────┘                  │
                  ▼                            │
         ┌───────────────┐                    │
         │ base_react.py │  ← 地基             │
         └───────┬───────┘                    │
                 │ 委托                        │ 调用
                 ▼                            │
           tool_executor.py  ←────────────────┘
                 │
                 ▼
           tool_retry_engine.py → 各category_tools.py
```

- **继承链**：Universal/Desktop → base_react.py（地基）
- **混入链**：RollbackMixin → 混入UniversalReactAgent
- **委托链**：base_react.py → tool_executor.py（Agent层向下委托）
- **调用链**：RollbackMixin.rollback() → tool_executor.py::execute_tool_with_unified_retry()

**Agent层**定义"什么时候调工具、调哪个工具"，**Tool层**负责"怎么把工具跑起来"。Agent层不碰工具实现，只通过`_execute_tool()`这个抽象方法向下层委托。
