  │
  ├─② AIServiceFactory.get_service() ─→ BaseAIService实例

  ├─③ register_task(task_id) ─→ _running_tasks注册

  │     │                    ─→ ToolManager.init_tools()
  │     │
  │     └─→ agent.run_stream() ─→ ReAct循环(Thought→Action→Observation→Final)
  │           │
  │           ├─ LLM调用: chat_with_tools_stream() ─→ parse_sse_stream()
  │           ├─ 响应解析: parse_react_response() ─→ 6-handler链

### 3.1 流程1: 主请求流 (chat_stream_v2)

#### 完整调用链

```
chat_stream_v2(request: ChatRequest)
  ├─ detect_intent(user_input)
  │    └─ route_with_fallback(user_input)
  │         └─ crss_scorer.detect_intent_v2(user_input)
  │              └─ _compute_intent_scores(command)
  │                   ├─ _match_keywords() [类型层]
  │                   ├─ ACTION_DEFINITIONS匹配 [动作层]
  │                   └─ 双维度合成 + 归一化
  ├─ AIServiceFactory.get_service()
  └─ StreamingResponse(generate())
       ├─ register_task(task_id)
       ├─ task_interrupt_check(task_id)
       ├─ task_pause_check(task_id)
       ├─ step_start() → StartStep → SSE
       ├─ run_sse_stream(intent_type, ...) → [核心SSE流]
       ├─ task_cancel_check_and_yield(task_id)
       └─ task_cleanup(task_id)
```

#### 核心伪代码

```python
async def chat_stream_v2(request):
    # ① 意图检测（generate外部，只执行一次）
    intent_type, source, confidence, candidates = await detect_intent(user_input)

    # ② 获取LLM服务
    ai_service = AIServiceFactory.get_service()

    async def generate():
        task_id = uuid4()
        try:
            await register_task(task_id, ai_service)
            # ③ 中断/暂停检查
            if await task_interrupt_check(task_id): yield interrupt_msg; return
            async for pause_event in task_pause_check(task_id): yield pause_event
            # ④ 发送start步骤
            async for event in step_start(...): yield event
            # ⑤ 运行SSE流（核心）
            async for sse_chunk in run_sse_stream(intent_type, ...):
                if await task_cancel_check_and_yield(task_id, ...): break
                yield sse_chunk
        except Exception as e:
            yield error_response
        finally:
            await task_cleanup(task_id, ...)

    return StreamingResponse(generate(), media_type="text/event-stream")
```

#### 关键数据结构

| 结构 | 类型 | 说明 |
|------|------|------|
| `_running_tasks` | `dict[str, dict]` | 全局任务注册表(status/cancelled/paused/ai_service) |
| `execution_steps` | `List[Dict]` | 当前会话所有步骤 |
| `agent_llm_holder` | `Dict[str, int]` | 跨生成器传递LLM调用次数 |
| `next_step` | `Callable[[], int]` | 步骤计数器闭包 |

### 3.5 流程5: SSE输出和持久化

#### 完整调用链

```
run_sse_stream(intent_type, llm_client, task_id, ...)
  ├─ AgentFactory.create(intent_type, ...) → agent
  ├─ agent.run_stream(task, context, task_id)
  └─ async for event in agent.run_stream(...):
       ├─ emit_only(event, current_content)
       │    ├─ format_agent_sse(step_obj) → SSE字符串
       │    └─ [final/chunk] 更新current_content
       ├─ save_execution_steps_to_db(session_id, execution_steps, current_content)
       └─ yield sse_data
```

#### Step类继承体系

```
ReasoningStep(ABC)    # 基类: step, timestamp, get_type(), to_dict()
  ├─ StartStep        # type="start"
  ├─ ThoughtStep      # type="thought", is_done=False
  ├─ ActionToolStep   # type="action_tool"
  ├─ ObservationStep  # type="observation"
  ├─ ChunkStep        # type="chunk", is_done=False
  ├─ FinalStep        # type="final", is_done=True
  ├─ ErrorStep        # type="error"
  └─ IncidentStep     # type="interrupted/paused/resumed/retrying"
```

#### SSE事件生命周期

```
① StartStep   → "data: {type:start,step:1,...}\n\n"
② ThoughtStep → "data: {type:thought,step:2,...}\n\n"
③ ActionToolStep → "data: {type:action_tool,step:3,...}\n\n"
④ ObservationStep → "data: {type:observation,step:4,...}\n\n"
⑤ ChunkStep   → "data: {type:chunk,step:5,...}\n\n"
⑥ FinalStep   → "data: {type:final,step:6,...}\n\n"
⑦ ErrorStep   → "data: {type:error,step:N,...}\n\n"
```

#### 持久化策略

- **保存时机**: 每个SSE chunk发送后立即保存
- **保存策略**: 全量替换——每次保存完整execution_steps列表
- **异常处理**: 会话不存在→标记`_INVALID_SESSION_IDS`跳过后续保存
