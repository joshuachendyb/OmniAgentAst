## 2 ReAct 标准流程

### 2.1 原始 ReAct 论文定义

```
Thought → Action → Observation → Thought → Action → Observation → ... → Final Answer
```

### 2.2 标准 Prompt 格式

```
Answer the following questions as best you can. You have access to tools provided.

Use the following format:

Question: the input question you must answer
Thought: you should always think about what to do
Action: the action to take
Action Input: the input to the action
Observation: the result of the action
... (this Thought/Action/Action Input/Observation can repeat N times)
Thought: I now know the final answer
Final Answer: the final answer to the original input question

Begin!
```

## 3 各框架字段对比

### 3.1 核心数据结构对比

| 概念 | LangChain | LlamaIndex | CrewAI | AutoGen | Haystack |
|------|-----------|------------|--------|---------|----------|
| **Thought** | `AgentAction.log` | `ActionReasoningStep.thought` | 内部封装 | Prompt 格式 | 内部封装 |
| **Action** | `AgentAction.tool` | `ActionReasoningStep.action` | 内部封装 | Prompt 格式 | 内部封装 |
| **Action Input** | `AgentAction.tool_input` | `ActionReasoningStep.action_input` | 内部封装 | Prompt 格式 | Tool 参数 |
| **Observation** | `intermediate_steps` 元组 | `ObservationReasoningStep.observation` | 内部封装 | Prompt 格式 | Tool 返回 |
| **Final Answer** | `AgentFinish.return_values` | `ResponseReasoningStep.response` | Task 输出 | Prompt 格式 | `last_message` |

### 3.2 LangChain 字段定义

```python
# AgentAction
class AgentAction(NamedTuple):
    tool: str                    # 工具名称
    tool_input: Union[str, dict]  # 工具输入
    log: str                      # LLM 原始思考日志

# AgentFinish
class AgentFinish(NamedTuple):
    return_values: dict          # 返回值 {"output": "..."}
    log: str                      # 最终日志

# 中间步骤
intermediate_steps: List[Tuple[AgentAction, str]]
# AgentAction + Observation (str) 配对
```

### 3.3 LlamaIndex 字段定义

```python
# ActionReasoningStep
@dataclass
class ActionReasoningStep(ReasoningStep):
    thought: str              # 思考内容
    action: str              # 工具名称
    action_input: dict       # 工具参数

# ObservationReasoningStep
@dataclass
class ObservationReasoningStep(ReasoningStep):
    observation: str         # 观察结果

# ResponseReasoningStep (Final Answer)
@dataclass
class ResponseReasoningStep(ReasoningStep):
    reasoning: str           # 完整推理
    response: str           # 最终回答
```

## 5 关键设计模式

### 5.1 Agent Loop 模式

**LangChain 实现**:

```python
def run(self, input_str: str, max_iterations: int = 50) -> dict:
    intermediate_steps = []
    inputs = {"input": input_str, "intermediate_steps": []}

    for _ in range(max_iterations):
        result = self.agent.invoke(inputs)

        if isinstance(result, AgentFinish):
            return {"output": result.return_values["output"]}

        # 执行工具
        observation = self._execute_tool(result)
        intermediate_steps.append((result, observation))
        inputs = {
            "input": input_str,
            "intermediate_steps": intermediate_steps,
        }

    return {"output": "Max iterations reached"}
```

**LlamaIndex Workflow 实现**:

```python
class ReActAgentWorkflow(Workflow):
    @step
    async def handle_llm_input(self, ctx, ev) -> ToolCallEvent | StopEvent:
        response = await self.llm.achat(chat_history)
        reasoning_step = self.output_parser.parse(response.message.content)
        
        if reasoning_step.is_done:
            return StopEvent(result={"response": reasoning_step.response})
        elif isinstance(reasoning_step, ActionReasoningStep):
            return ToolCallEvent(tool_calls=[...])
```

## 8 omniAgent 参考建议

### 8.1 字段设计建议

基于 LangChain/LlamaIndex 的最佳实践：

```python
# Thought + Action
@dataclass
class ThoughtActionStep:
    type: str = "thought_action"
    thought: str = ""
    action: str = ""
    action_input: dict = field(default_factory=dict)
    timestamp: int = 0

# Observation
@dataclass
class ObservationStep:
    type: str = "observation"
    content: str = ""
    timestamp: int = 0

# Final Answer
@dataclass
class FinalStep:
    type: str = "final"
    content: str = ""
    timestamp: int = 0
```

### 8.2 Loop 实现建议

```python
class ReActLoop:
    def __init__(self, llm, tools):
        self.llm = llm
        self.tools = {t.name: t for t in tools}
        self.parser = ReActOutputParser()
    
    async def run(self, task: str) -> List[dict]:
        steps = []
        prompt = self._build_prompt(task)
        
        while len(steps) < MAX_ITERATIONS:
            # 1. 调用 LLM
            response = await self.llm.invoke(prompt, stop=["\nObservation:"])
            
            # 2. 解析输出
            step = self.parser.parse(response.content)
            
            if step.type == "final":
                steps.append(step.to_dict())
                break
            
            elif step.type == "thought_action":
                steps.append(step.to_dict())
                
                # 3. 执行工具
                observation = await self._execute_tool(step.action, step.action_input)
                steps.append({"type": "observation", "content": observation})
                
                # 4. 更新 prompt
                prompt += f"\n{step.to_text()}\nObservation: {observation}"
        
        return steps
```

### 8.3 关键实现要点

| 要点 | 说明 |
|------|------|
| **Stop Sequence** | 使用 `\nObservation:` 防止 LLM 幻觉 |
| **Output Parser** | 正则解析 Thought/Action/Action Input |
| **中间步骤管理** | 保留 (Action, Observation) 配对 |
| **上下文裁剪** | 保留最近 N 步防止 overflow |
| **工具注册** | 统一的工具定义格式 |
