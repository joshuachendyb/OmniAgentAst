## 1. 工具系统概览

### 1.1 系统架构
Omni系统采用前后端分离架构，工具系统作为Agent核心能力的一部分，主要负责：
- 文件操作（读写、搜索、管理）
- 命令执行
- 系统信息获取
- 报告生成

### 1.2 技术栈
- **后端框架**: FastAPI + Uvicorn
- **Python版本**: >=3.11
- **数据验证**: Pydantic 2.5.3
- **工具架构**: 基于ReAct模式的Agent工具调用

### 1.3 核心组件
1. **工具注册系统** - 管理工具定义和Schema
2. **工具解析器** - 解析LLM输出中的工具调用
3. **工具执行器** - 执行工具并处理结果
4. **安全服务** - 文件操作权限控制
5. **会话管理** - 工具调用上下文管理

### 2.2 工具调用流程分析

#### 2.2.1 完整调用链

```
用户请求 → Agent.run_stream() → LLM生成响应 → parse_react_response()解析

```

##### 节点2：LLM响应解析 (`react_output_parser.py:47-73`)

**函数**: `parse_react_response(output: str) -> Dict[str, Any]`

**输入参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| output | str | LLM原始响应文本 |

**返回格式**:
```python
{
    "type": "action|answer|implicit|thought_only|parse_error",
    "thought": "思考内容",
    "content": "思考内容（兼容字段）",
    "tool_name": "工具名|finish",
    "tool_params": {"param1": "value1"},  # 或 tool_params
    "params": {"param1": "value1"},        # 兼容字段
    "response": "最终回答",
    "error": "解析错误信息"  # 仅parse_error时
}
```

**解析策略**（五级降级）:
1. JSON格式解析 → action/answer
2. 中英文关键词匹配 → action
3. 工具名兜底 → action
4. 隐式回答检测 → implicit
5. 最终兜底 → thought_only

**异常处理**:
- 空输入 → 返回 `type: "parse_error"`
- 非字符串 → 返回 `type: "parse_error"`
- JSON解析失败 → 降级到关键词匹配

##### 节点3：工具执行 (`file_react.py:330-340` → `tool_executor.py:55-158`)

**函数**: `async def _execute_tool(action: str, params: Dict) -> Dict`

**输入参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| action | str | 工具名称（如"read_file"）|
| params | Dict | 工具参数字典 |

**执行器execute()流程**:

```python
# 1. 检查finish命令（直接返回）
if action == "finish":
    return {"status": "success", "summary": "Task completed", ...}

# 2. 检查工具是否存在
if action not in self.available_tools:
    return {"status": "error", "error": f"Unknown tool: {action}", ...}

# 3. 参数规范化（别名映射）
normalized_params = self._normalize_params(action, params)

# 4. 参数验证（反射）
self._validate_params(tool_func, normalized_params)

# 5. 超时控制
result = await asyncio.wait_for(tool_func(**normalized_params), timeout=timeout)

# 6. 结果格式化
return self._format_result(result, action)
```

**返回格式**:
```python
{
    "status": "success|error|timeout|permission_denied",
    "summary": "执行摘要",
    "data": {...},      # 实际数据
    "error": "错误信息",  # 仅error/timeout
    "retry_count": 0     # 重试次数
}
```

**超时配置** (`TOOL_TIMEOUTS`):
| 工具 | 超时(秒) |
|------|---------|
| read_file | 30 |
| write_file | 30 |
| delete_file | 30 |
| list_directory | 10 |
| search_files | 30 |
| search_file_content | 60 |
| generate_report | 120 |

**异常处理**:
- `KeyError` → 返回 `status: "error"`, error: "缺少参数"
- `PermissionError` → 返回 `status: "permission_denied"`
- `FileNotFoundError` → 返回 `status: "error"`, error: "文件不存在"
- `asyncio.TimeoutError` → 返回 `status: "timeout"`
- 其他Exception → 返回 `status: "error"`, error: 具体错误信息

#### 2.3.3 工具执行流程

**位置**: `backend/app/services/agent/tool_executor.py`

**核心代码**:
```python
# ToolExecutor.execute() 关键逻辑
async def execute(self, action: str, action_input: Dict[str, Any]) -> Dict[str, Any]:
    # 1. 参数验证（反射检查必需参数）
    # 2. 参数别名映射（硬编码映射表）
    # 3. 超时控制（硬编码超时配置）
    # 4. 异常处理（分类处理）
    # 5. 结果格式化（统一格式）
```

**执行数据流**:
```
execute(action, action_input)
       ↓
[Step 1] 检查 action == "finish"
       ↓ 有 → 直接返回成功
       ↓ 无
[Step 2] 检查工具是否存在
       ↓ 不存在 → 返回 error
       ↓ 存在
[Step 3] 参数规范化 (_normalize_params)
       ↓
[Step 4] 参数验证 (_validate_params)
       ↓ 失败 → 返回 error
       ↓ 成功
[Step 5] 异步执行 (asyncio.wait_for)
```

```python
def _get_default_allowed_paths() -> List[Path]:
    """获取默认允许的路径列表"""
    paths = [
        Path.home(),  # 用户主目录
        Path("/tmp"),  # Linux临时目录
        Path("/var/tmp"),  # Linux临时目录
    ]
    
    # Windows盘符（A-J）
    if os.name == 'nt':
        for letter in 'ABCDEFGHIJ':
            drive = Path(f"{letter}:/")
            if drive.exists():
                paths.append(drive)
    
    return paths

ALLOWED_PATHS = _get_default_allowed_paths()
```

**安全特性**:
| 特性 | 说明 |

| **主目录** | 自动允许用户主目录 |
| **临时目录** | 允许/tmp和/var/tmp |

**白名单检查逻辑**:
```python
def _check_path_in_allowed(path: Path) -> bool:
    """检查路径是否在白名单内"""
    try:
        path = path.resolve()
        for allowed in ALLOWED_PATHS:
            if path.is_relative_to(allowed):
                return True
        return False
    except Exception:
        return False
```

#### 6.2.1 工具注册机制重构
**问题**: 当前使用全局字典`_TOOL_REGISTRY`，缺乏类型安全和生命周期管理

**改进方案**:
```python
# backend/app/services/tools/registry.py
from typing import Dict, List, Optional, Type, Any
from dataclasses import dataclass
from datetime import datetime
from enum import Enum

class ToolCategory(Enum):
    FILE = "file"
    DATABASE = "database"
    NETWORK = "network"
    SYSTEM = "system"
    DESKTOP = "desktop"

@dataclass
class ToolMetadata:
    name: str
    description: str
    category: ToolCategory
    version: str = "1.0.0"
    author: str = ""
    dependencies: List[str] = None
    input_schema: Dict[str, Any] = None
    output_schema: Dict[str, Any] = None
    examples: List[Dict[str, Any]] = None
    created_at: datetime = None
    updated_at: datetime = None
    
    def __post_init__(self):
        if self.dependencies is None:
            self.dependencies = []
        if self.examples is None:
            self.examples = []
        if self.created_at is None:
            self.created_at = datetime.now()
        if self.updated_at is None:
            self.updated_at = self.created_at

class ToolRegistry:
    """类型安全的工具注册表"""
    
    def __init__(self):
        self._tools: Dict[str, ToolMetadata] = {}
        self._categories: Dict[ToolCategory, List[str]] = {}
        self._implementations: Dict[str, callable] = {}
    
    def register(
        self,
        name: str,
        description: str,
        category: ToolCategory,
        implementation: callable,
        version: str = "1.0.0",
        dependencies: Optional[List[str]] = None,
        input_schema: Optional[Dict] = None,
        output_schema: Optional[Dict] = None,
        examples: Optional[List[Dict]] = None
    ) -> None:
        """注册工具"""
        if name in self._tools:
            raise ValueError(f"Tool '{name}' already registered")
        
        # 验证依赖关系
        if dependencies:
            missing = [dep for dep in dependencies if dep not in self._tools]
            if missing:
                raise ValueError(f"Missing dependencies: {missing}")
        
        # 创建工具元数据
        metadata = ToolMetadata(
            name=name,
            description=description,
            category=category,
            version=version,
            dependencies=dependencies or [],
            input_schema=input_schema,
            output_schema=output_schema,
            examples=examples or []
        )
        
        # 注册工具
        self._tools[name] = metadata
        self._implementations[name] = implementation
        
        # 更新分类索引
        if category not in self._categories:
            self._categories[category] = []
        self._categories[category].append(name)
        
        # 更新工具更新时间
        metadata.updated_at = datetime.now()
    
    def get_tool(self, name: str) -> Optional[ToolMetadata]:
        """获取工具元数据"""
        return self._tools.get(name)
    
    def get_implementation(self, name: str) -> Optional[callable]:
        """获取工具实现"""
        return self._implementations.get(name)
    
    def list_tools(
        self, 
        category: Optional[ToolCategory] = None,
        include_metadata: bool = False
    ) -> List[Union[str, ToolMetadata]]:
        """列出工具"""
        if category:
            tool_names = self._categories.get(category, [])
        else:
            tool_names = list(self._tools.keys())
        
        if include_metadata:
            return [self._tools[name] for name in tool_names]
        return tool_names
    
    def unregister(self, name: str) -> bool:
        """注销工具"""
        if name not in self._tools:
            return False
        
        metadata = self._tools[name]
        
        # 从分类索引中移除
        category_tools = self._categories.get(metadata.category, [])
        if name in category_tools:
            category_tools.remove(name)
        
        # 移除工具
        del self._tools[name]
        del self._implementations[name]
        
        return True
```

from abc import ABC, abstractmethod

@dataclass
class ParseResult:

    type: str  # "action", "answer", "implicit", "thought_only", "parse_error"

class BaseParser(ABC):
    """解析器基类"""
    
    @abstractmethod
    def can_parse(self, output: str) -> bool:
        """判断是否能解析此输出"""
        pass
    
    @abstractmethod
    def parse(self, output: str) -> ParseResult:

# json_parser.py
import json
import re
from .base_parser import BaseParser, ParseResult

class JsonParser(BaseParser):
    """JSON格式解析器"""
    
    def can_parse(self, output: str) -> bool:
        # 检查是否包含JSON对象
        output = output.strip()
        return (output.startswith('{') and output.endswith('}')) or \
               (output.startswith('```json') and '```' in output)
    
    def parse(self, output: str) -> ParseResult:
        try:
            # 清理输出中的代码块标记
            cleaned = output.strip()
            if cleaned.startswith('```json'):
                cleaned = cleaned[7:]  # 移除 ```json
            if cleaned.endswith('```'):
                cleaned = cleaned[:-3]  # 移除 ```
            cleaned = cleaned.strip()
            
            # 解析JSON
            data = json.loads(cleaned)
            
            # 提取工具信息
            tool_name = data.get("tool_name") or data.get("action")
            tool_params = data.get("tool_params") or data.get("action_input") or {}
            
            # 判断类型
            if tool_name == "finish" or "answer" in data:
                return ParseResult(
                    success=True,
                    type="answer",
                    thought=data.get("thought") or data.get("content", ""),
                    response=data.get("response") or data.get("answer", "")
                )
            elif tool_name:
                return ParseResult(
                    success=True,
                    type="action",
                    tool_name=tool_name,
                    tool_params=tool_params,
                    thought=data.get("thought") or data.get("content", "")
                )
            else:
                return ParseResult(
                    success=True,
                    type="implicit",
                    thought=data.get("thought") or data.get("content", ""),
                    response=output
                )
        except json.JSONDecodeError as e:
            return ParseResult(
                success=False,
                type="parse_error",
                error=f"JSON解析失败: {str(e)}"
            )

    def parse(self, output: str) -> ParseResult:
        """使用解析器链解析输出"""
        if not output or not isinstance(output, str):
            return ParseResult(
                success=False,
                type="parse_error",
                error="Empty or non-string response"
            )
        
        # 按优先级尝试解析
        for parser in self.parsers:
            if parser.can_parse(output):
                result = parser.parse(output)
                if result.success:
                    return result
        
        # 所有解析器都失败
        return ParseResult(
            success=False,
            type="parse_error",
            error="无法解析LLM响应"
        )

#### 6.2.3 执行器增强
**问题**: 当前执行器缺少重试机制（`retry_count`字段存在但未实现），错误分类不够细化

#### 过度设计审视

> 原方案引入了熔断器（CircuitBreaker）、指标收集器（MetricsCollector）、重试策略（RetryPolicy）三大组件，但对当前系统而言存在过度设计风险：

| 组件 | 原方案 | 当前适用性 | 问题 |
|------|--------|-----------|------|
| **CircuitBreaker** | 失败5次后熔断60秒 | ❌ 不适用 | 文件工具失败多为"文件不存在"等业务错误，熔断会阻止后续对其他文件的正常请求；单实例无下游服务保护需求 |
| **MetricsCollector** | 推送到Prometheus | ❌ 不适用 | 7.3节已明确"当前不需要监控和可观测性"，日志系统已足够 |
| **RetryPolicy** | 指数退避重试 | ✅ 适用 | 超时和网络错误确实需要重试，这是当前 `retry_count` 字段的应有之义 |
| **ErrorClassifier** | 错误类型枚举 | ✅ 适用 | 当前错误处理是散落的if-elif，统一分类有助于重试决策和日志分析 |
| **ParamValidator** | Pydantic Schema验证 | ⚠️ 可选 | 已有Pydantic模型定义，但当前inspect方式可工作，优先级低 |

**精简方案**：只实现 **重试机制 + 错误分类**，移除熔断器和指标收集器。

**改进方案（精简版）**:
```python
# backend/app/services/agent/executor/

# base_executor.py
from abc import ABC, abstractmethod
from typing import Dict, Any, Optional
from dataclasses import dataclass
from datetime import datetime
import asyncio

@dataclass
class ToolCall:
    name: str
    params: Dict[str, Any]
    session_id: str
    user_id: Optional[str] = None
    timestamp: datetime = None
    
    def __post_init__(self):
        if self.timestamp is None:
            self.timestamp = datetime.now()

@dataclass
class ExecutionResult:
    status: str  # "success", "error", "timeout", "permission_denied"
    summary: str
    data: Optional[Any] = None
    error: Optional[str] = None
    execution_time_ms: int = 0
    retry_count: int = 0
    metadata: Optional[Dict[str, Any]] = None

class BaseExecutor(ABC):
    """执行器基类"""
    
    @abstractmethod
    async def execute(self, tool_call: ToolCall) -> ExecutionResult:
        """执行工具调用"""
        pass
    
    @abstractmethod
    def validate(self, tool_call: ToolCall) -> bool:
        """验证工具调用"""
        pass

# tool_executor.py
import inspect
from typing import Dict, Any, Callable
from .base_executor import BaseExecutor, ToolCall, ExecutionResult
from .retry_policy import RetryPolicy
from .error_classifier import ErrorClassifier, ErrorType

class EnhancedToolExecutor(BaseExecutor):
    """增强版工具执行器（精简版：重试+错误分类，无熔断/指标）"""
    
    def __init__(
        self,
        tools: Dict[str, Callable],
        config: Optional[Dict[str, Any]] = None
    ):
        self.tools = tools
        self.config = config or {}
        
        # 初始化重试策略
        self.retry_policy = RetryPolicy(
            max_retries=self.config.get("max_retries", 3),
            backoff_factor=self.config.get("backoff_factor", 2.0),
            retryable_errors=self.config.get("retryable_errors", ["timeout", "network_error"])
        )
        
        # 初始化错误分类器
        self.error_classifier = ErrorClassifier()
        
        # 参数别名映射（从配置加载）
        self.param_aliases = self.config.get("param_aliases", {})
        
        # 超时配置（从配置加载）
        self.timeouts = self.config.get("timeouts", {
            "default": 30,
            "read_file": 30,
            "search_file_content": 60,
            "execute_command": 120
        })
    
    async def execute(self, tool_call: ToolCall) -> ExecutionResult:
        """执行工具调用（精简增强版）"""
        start_time = datetime.now()
        
        # 1. 验证工具调用
        if not self.validate(tool_call):
            return ExecutionResult(
                status="error",
                summary=f"Invalid tool call: {tool_call.name}",
                error="Tool validation failed"
            )
        
        # 2. 获取工具函数
        tool_func = self.tools.get(tool_call.name)
        if not tool_func:
            return ExecutionResult(
                status="error",
                summary=f"Unknown tool: {tool_call.name}",
                error=f"Available tools: {list(self.tools.keys())}"
            )
        
        # 4. 参数预处理
        normalized_params = self._normalize_params(tool_call.name, tool_call.params)
        
        # 5. 参数验证
        validation_result = self._validate_params(tool_func, normalized_params)
        if not validation_result.valid:
            return ExecutionResult(
                status="error",
                summary=f"Parameter validation failed: {validation_result.errors}",
                error="Invalid parameters"
            )
        
        # 5. 执行工具（支持重试）
        result = await self._execute_with_retry(
            tool_call.name,
            tool_func,
            normalized_params,
            tool_call.session_id
        )
        
        # 6. 记录执行时间并返回
        execution_time_ms = int((datetime.now() - start_time).total_seconds() * 1000)
        result.execution_time_ms = execution_time_ms
        
        # 日志记录（替代MetricsCollector，使用已有的logger）
        import logging
        logger = logging.getLogger(__name__)
        if result.status == "success":
            logger.info(f"Tool '{tool_call.name}' executed in {execution_time_ms}ms")
        else:
            logger.warning(f"Tool '{tool_call.name}' failed: {result.error}, retries={result.retry_count}")
        
        return result
    
    async def _execute_with_retry(
        self,
        tool_name: str,
        tool_func: Callable,
        params: Dict[str, Any],
        session_id: str
    ) -> ExecutionResult:
        """带重试的执行（使用ErrorClassifier统一分类）"""
        timeout = self.timeouts.get(tool_name, self.timeouts["default"])
        
        for attempt in range(self.retry_policy.max_retries + 1):
            try:
                # 设置超时
                result = await asyncio.wait_for(
                    tool_func(**params),
                    timeout=timeout
                )
                
                # 格式化结果
                return self._format_result(result, tool_name)
                
            except Exception as e:
                # 使用错误分类器统一分类
                error_type = self.error_classifier.classify(e)
                
                # 不可重试的错误：直接返回
                if not error_type.is_retryable():
                    return ExecutionResult(
                        status=error_type.to_status(),
                        summary=f"{error_type.description}: {str(e)}",
                        error=str(e),
                        retry_count=attempt
                    )
                
                # 可重试的错误：达到最大重试次数则返回失败
                if attempt == self.retry_policy.max_retries:
                    return ExecutionResult(
                        status=error_type.to_status(),
                        summary=f"{error_type.description}: {str(e)} (after {attempt} retries)",
                        error=str(e),
                        retry_count=attempt
                    )
                # 重试
                await asyncio.sleep(self.retry_policy.get_backoff(attempt))
    
    def _normalize_params(self, tool_name: str, params: Dict[str, Any]) -> Dict[str, Any]:
        """参数规范化（支持动态配置）"""
        normalized = params.copy()
        
        # 应用参数别名映射
        aliases = self.param_aliases.get(tool_name, {})
        for wrong_name, correct_name in aliases.items():
            if wrong_name in normalized and correct_name not in normalized:
                normalized[correct_name] = normalized[wrong_name]
                del normalized[wrong_name]
        
        return normalized
    
    def _validate_params(self, tool_func: Callable, params: Dict[str, Any]) -> ValidationResult:
        """参数验证（使用Pydantic Schema）"""
        # 获取函数签名
        sig = inspect.signature(tool_func)
        
        # 检查必需参数
        required_params = [
            p.name for p in sig.parameters.values()
            if p.default == inspect.Parameter.empty and p.name != 'self'
        ]
        
        missing = [p for p in required_params if p not in params]
        if missing:
            return ValidationResult(
                valid=False,
                errors=f"Missing required parameters: {missing}"
            )
        
        # TODO: 使用Pydantic Schema进行类型验证
        return ValidationResult(valid=True)
    
    def _format_result(self, result: Any, tool_name: str) -> ExecutionResult:
        """格式化结果"""
        if isinstance(result, dict) and "status" in result:
            # 已经是标准格式
            return ExecutionResult(
                status=result.get("status", "success"),
                summary=result.get("summary", ""),
                data=result.get("data"),
                error=result.get("error"),
                retry_count=result.get("retry_count", 0)
            )
        else:
            # 转换为标准格式
            return ExecutionResult(
                status="success",
                summary=f"Tool '{tool_name}' executed successfully",
                data=result
            )
```

```python
from pydantic import BaseModel, ValidationError
from typing import Type

class ValidationResult:
    def __init__(self, valid: bool, errors: str = None):
        self.valid = valid
        self.errors = errors

class ParamValidator:
    """参数验证器 - 使用Pydantic Schema"""
    
    def __init__(self):
        self._schema_cache: Dict[str, Type[BaseModel]] = {}
    
    def register_schema(self, tool_name: str, schema_class: Type[BaseModel]):
        """注册工具的Pydantic Schema"""
        self._schema_cache[tool_name] = schema_class
    
    def validate(self, tool_name: str, params: Dict[str, Any]) -> ValidationResult:
        """验证参数"""
        if tool_name not in self._schema_cache:
            # 无Schema，使用旧的inspect方式
            return ValidationResult(valid=True)
        
        schema_class = self._schema_cache[tool_name]
        try:
            schema_class(**params)
            return ValidationResult(valid=True)
        except ValidationError as e:
            errors = "; ".join([f"{err['loc']}: {err['msg']}" for err in e.errors()])
            return ValidationResult(valid=False, errors=errors)
```

```python
from enum import Enum

class ErrorType(Enum):
    TIMEOUT = "timeout"              # 可重试
    PERMISSION_DENIED = "permission_denied"  # 不可重试
    FILE_NOT_FOUND = "file_not_found"      # 不可重试
    INVALID_PARAMS = "invalid_params"      # 不可重试
    TOOL_NOT_FOUND = "tool_not_found"      # 不可重试
    NETWORK_ERROR = "network_error"        # 可重试
    UNKNOWN = "unknown"              # 默认不可重试
    
    @property
    def description(self) -> str:
        """错误类型中文描述"""
        descriptions = {
            "timeout": "执行超时",
            "permission_denied": "权限拒绝",
            "file_not_found": "文件不存在",
            "invalid_params": "参数无效",
            "tool_not_found": "工具不存在",
            "network_error": "网络错误",
            "unknown": "未知错误"
        }
        return descriptions.get(self.value, "未知错误")
    
    def is_retryable(self) -> bool:
        """是否可重试"""
        return self in (ErrorType.TIMEOUT, ErrorType.NETWORK_ERROR)
    
    def to_status(self) -> str:
        """转换为ExecutionResult的status字段"""
        mapping = {
            ErrorType.TIMEOUT: "timeout",
            ErrorType.PERMISSION_DENIED: "permission_denied",
            ErrorType.FILE_NOT_FOUND: "error",
            ErrorType.INVALID_PARAMS: "error",
            ErrorType.TOOL_NOT_FOUND: "error",
            ErrorType.NETWORK_ERROR: "error",
            ErrorType.UNKNOWN: "error"
        }
        return mapping.get(self, "error")

class ErrorClassifier:
    """错误分类器"""
    
    @staticmethod
    def classify(exception: Exception) -> ErrorType:
        if isinstance(exception, asyncio.TimeoutError):
            return ErrorType.TIMEOUT
        elif isinstance(exception, PermissionError):
            return ErrorType.PERMISSION_DENIED
        elif isinstance(exception, FileNotFoundError):
            return ErrorType.FILE_NOT_FOUND
        elif isinstance(exception, (ConnectionError, OSError)):
            # 网络相关错误
            return ErrorType.NETWORK_ERROR
        else:
            error_msg = str(exception)
            if "Unknown tool" in error_msg:
                return ErrorType.TOOL_NOT_FOUND
            return ErrorType.UNKNOWN
```

```yaml
# backend/app/services/tools/config.py
import yaml
from pathlib import Path
from typing import Dict, Any

class ToolConfig:
    """工具配置管理器"""
    
    def __init__(self, config_path: Optional[str] = None):
        self.config_path = config_path or "config/tools.yaml"
        self._config = self._load_config()
    
    def _load_config(self) -> Dict[str, Any]:
        """加载配置"""
        config_file = Path(self.config_path)
        if not config_file.exists():
            return self._get_default_config()
        
        with open(config_file, 'r', encoding='utf-8') as f:
            config = yaml.safe_load(f)
        
        # 环境变量替换
        config = self._replace_env_vars(config)
        return config
    
    def _replace_env_vars(self, config: Dict[str, Any]) -> Dict[str, Any]:
        """替换环境变量"""
        import os
        
        def replace(obj):
            if isinstance(obj, str) and obj.startswith("${") and obj.endswith("}"):
                var_name = obj[2:-1]
                return os.getenv(var_name, obj)
            elif isinstance(obj, dict):
                return {k: replace(v) for k, v in obj.items()}
            elif isinstance(obj, list):
                return [replace(item) for item in obj]
            else:
                return obj
        
        return replace(config)
    
    def _get_default_config(self) -> Dict[str, Any]:
        """获取默认配置"""
        return {
            "tools": {
                "timeouts": {"default": 30},
                "aliases": {},
                "retry": {"max_retries": 3},
                "security": {"sandbox_enabled": False},
                "monitoring": {"enabled": False},
                "registry": {"auto_discover": False}
            }
        }
    
    def get(self, key: str, default: Any = None) -> Any:
        """获取配置值"""
        keys = key.split('.')
        value = self._config
        for k in keys:
            if isinstance(value, dict):
                value = value.get(k)
            else:
                return default
            if value is None:
                return default
        return value
    
    def reload(self) -> None:
        """重新加载配置"""
        self._config = self._load_config()
```

```python
class ToolConfig:
    # ... 原有代码 ...
    
    def get(self, key: str, default: Any = None) -> Any:
        """获取配置值（带降级）"""
        keys = key.split('.')
        value = self._config
        for k in keys:
            if isinstance(value, dict):

                return default

                if default is not None:
                    logger.debug(f"配置项 '{key}' 不存在，使用默认值: {default}")
                return default
        return value

        """获取工具超时（带降级链）"""

        return self.get(
            f"tools.timeouts.{tool_name}",
            self.get("tools.timeouts.default", 30)
```

**4. 环境变量替换错误处理**

```python
def _replace_env_vars(self, config: Dict[str, Any]) -> Dict[str, Any]:
    """替换环境变量（增加错误处理）"""
    import os
    
    def replace(obj):
        if isinstance(obj, str) and obj.startswith("${") and obj.endswith("}"):
            var_name = obj[2:-1]
            value = os.getenv(var_name)
            if value is None:
                # 环境变量未设置：记录警告，保留原始占位符
                logger.warning(f"环境变量 '${{{var_name}}}' 未设置，配置项保留原值")
                return obj  # 保留 ${VAR_NAME} 原值，而非替换为None
            return value
        elif isinstance(obj, dict):
            return {k: replace(v) for k, v in obj.items()}
        elif isinstance(obj, list):
            return [replace(item) for item in obj]
        else:
            return obj
    
    return replace(config)
```

**配置安全机制总结**：

| 机制 | 解决的问题 | 实现方式 |
|------|-----------|---------|
| **Schema验证** | 配置写错（如超时为负数）导致运行时异常 | Pydantic模型验证，失败时降级到默认配置 |
| **热重载原子性** | 重载一半时新请求读到半新半旧配置 | 先加载到临时变量，再原子性赋值替换 |
| **降级策略** | 配置项缺失导致KeyError | `get()` 方法带默认值 + `get_timeout()` 降级链 |
| **环境变量容错** | 环境变量未设置导致None值 | 未设置时保留原占位符并记录警告 |
