## 方案2：使用Vite自动分割（更简单）✅ 已采用
// 把 manualChunks: undefined 改成空对象
manualChunks: {},
// 方法2：使用Vite自动分割
    rollupOptions: {
      output: {
        manualChunks: {},  // 空对象，让Vite自动按node_modules分割
      },
    },
让Vite自动按node_modules分割，不手动指定。

---

【实测记录 - 2026-04-13】
当前配置：完全移除 manualChunks，让Vite默认自动处理
