## 1. 文档概述

### 1.1 背景
当前Omni系统的工具实现主要针对文件操作场景，缺乏对多类型工具的统一支持和管理机制。随着业务需求增长，需要支持数据库操作、网络请求、系统命令、AI模型调用等多种工具类型，并实现工具间的组合和编排。

### 1.2 范围
本文档涵盖Omni系统多工具支持的扩展设计，包括：
- 工具类型系统设计
- 工具注册和管理机制
- 工具组合和编排引擎
- 安全和控制机制
- 配置和部署方案

### 1.3 目标读者
- 系统架构师
- 后端开发工程师
- 安全工程师
- 运维工程师

### 1.4 术语定义
| 术语 | 定义 |
|------|------|
| Tool | 工具，执行特定功能的可调用单元 |
| Tool Type | 工具类型，按功能分类的工具类别 |
| Tool Chain | 工具链，按顺序执行的工具序列 |
| Workflow | 工作流，包含条件判断和依赖关系的工具组合 |
| Sandbox | 沙箱，隔离的工具执行环境 |

## 2. 设计目标

### 2.1 核心目标
1. **支持多类型工具**: 文件操作、数据库查询、网络请求、系统命令、AI模型等
2. **统一管理接口**: 所有工具使用统一的注册、发现、调用接口
3. **工具组合编排**: 支持工具链和工作流，实现复杂业务逻辑
4. **安全可控**: 细粒度的权限控制、资源限制、执行隔离
5. **易于扩展**: 插件化架构，支持动态加载新工具

### 2.2 非功能性目标
1. **性能**: 工具调用延迟<100ms，支持100+并发调用
2. **可用性**: 系统可用性>99.9%，支持故障转移
3. **安全性**: 完整的权限控制和审计日志
4. **可维护性**: 代码模块化，配置驱动，易于调试
5. **可观测性**: 完整的监控、日志、追踪

### 2.3 成功标准
1. 支持至少8种工具类型
2. 工具注册时间<1秒
3. 工具调用成功率>99.5%
4. 支持工具链嵌套深度≥5层
5. 安全策略覆盖所有工具操作

### 3.2 组件职责

#### 3.2.1 工具网关层 (Tool Gateway)
- **入口点**: 提供统一的REST/gRPC接口
- **认证授权**: JWT验证、API密钥、OAuth2.0
- **限流熔断**: 基于令牌桶的限流、熔断器模式
- **请求路由**: 根据工具类型路由到不同的执行器

#### 3.2.2 工具管理层 (Tool Manager)
- **注册中心**: 工具注册、发现、注销
- **依赖管理**: 工具间依赖关系解析
- **配置管理**: 工具配置的热加载
- **版本管理**: 多版本工具共存和切换

#### 3.2.3 工具编排层 (Tool Orchestrator)
- **工作流引擎**: DAG（有向无环图）执行引擎
- **条件执行**: 基于上下文的条件判断
- **错误处理**: 重试、回滚、补偿事务
- **状态管理**: 工作流状态持久化

#### 3.2.4 工具执行层 (Tool Executor)
- **执行引擎**: 异步执行工具调用
- **资源管理**: CPU、内存、网络资源限制
- **超时控制**: 可配置的超时策略
- **监控指标**: 执行时间、成功率、错误率

#### 3.2.5 工具实现层 (Tool Implementations)
- **标准接口**: 所有工具实现统一的Tool接口
- **类型分类**: 按功能分类的工具实现
- **插件机制**: 支持动态加载第三方工具

#### 3.2.6 安全层 (Security Layer)
- **权限模型**: 基于角色的访问控制（RBAC）
- **执行隔离**: Docker/容器化沙箱环境
- **资源配额**: 内存、CPU、磁盘、网络限制
- **审计追踪**: 完整的操作日志和审计追踪

### 3.3 数据流
```
用户请求 → 网关层（认证/限流） → 管理层（工具发现） → 编排层（工作流解析）
    ↓
执行层（工具执行） → 实现层（具体工具） → 安全层（权限/资源检查）
    ↓
返回结果 ← 编排层（结果聚合） ← 执行层（结果格式化）
```

### 3.4 技术选型
| 组件 | 技术选型 | 理由 |
|------|----------|------|
| 网关层 | FastAPI + Nginx | 高性能、异步支持、中间件丰富 |
| 管理层 | Redis + PostgreSQL | 高可用、持久化、事务支持 |
| 编排层 | Apache Airflow + Celery | 成熟的DAG引擎、任务调度 |
| 执行层 | asyncio + multiprocessing | 异步IO、进程隔离 |
| 安全层 | Docker + cgroups | 容器化隔离、资源限制 |
| 监控 | Prometheus + Grafana | 指标收集、可视化 |
| 日志 | ELK Stack (Elasticsearch, Logstash, Kibana) | 日志聚合、搜索、分析 |
| 追踪 | Jaeger + OpenTelemetry | 分布式追踪、性能分析 |

## 4. 工具类型系统

### 4.1 工具类型定义

```python
# backend/app/services/tools/types.py
from enum import Enum
from typing import Dict, Any, List, Optional, Set
from dataclasses import dataclass, field
from datetime import datetime
from pydantic import BaseModel, Field, validator

class ToolType(str, Enum):
    """工具类型枚举"""
    FILE_OPERATION = "file_operation"      # 文件操作
    DATABASE_QUERY = "database_query"      # 数据库查询
    NETWORK_REQUEST = "network_request"    # 网络请求
    SYSTEM_COMMAND = "system_command"      # 系统命令
    DESKTOP_AUTOMATION = "desktop_automation"  # 桌面自动化
    AI_MODEL = "ai_model"                  # AI模型调用
    DATA_PROCESSING = "data_processing"    # 数据处理
    VISUALIZATION = "visualization"        # 可视化
    CUSTOM = "custom"                      # 自定义工具

class ToolCapability(str, Enum):
    """工具能力枚举"""
    READ = "read"              # 读取数据
    WRITE = "write"            # 写入数据
    EXECUTE = "execute"        # 执行命令
    QUERY = "query"            # 查询数据
    TRANSFORM = "transform"    # 数据转换
    ANALYZE = "analyze"        # 数据分析
    VISUALIZE = "visualize"    # 数据可视化
    NOTIFY = "notify"          # 发送通知
    SCHEDULE = "schedule"      # 定时调度

class ToolPermission(str, Enum):
    """工具权限枚举"""
    READ_FILE = "read_file"                # 读取文件
    WRITE_FILE = "write_file"              # 写入文件
    DELETE_FILE = "delete_file"            # 删除文件
    EXECUTE_COMMAND = "execute_command"    # 执行命令
    NETWORK_ACCESS = "network_access"      # 网络访问
    DATABASE_ACCESS = "database_access"    # 数据库访问
    SYSTEM_INFO = "system_info"            # 系统信息
    USER_DATA = "user_data"                # 用户数据
    ADMIN = "admin"                        # 管理员权限

@dataclass
class ToolRequirements:
    """工具运行要求"""
    min_memory_mb: int = 128              # 最小内存（MB）
    max_memory_mb: int = 1024             # 最大内存（MB）
    min_cpu_cores: float = 0.5            # 最小CPU核心数
    max_cpu_cores: float = 2.0            # 最大CPU核心数
    timeout_seconds: int = 30              # 超时时间（秒）
    requires_network: bool = False         # 是否需要网络
    requires_disk: bool = False            # 是否需要磁盘
    requires_gpu: bool = False             # 是否需要GPU
    supported_platforms: List[str] = field(default_factory=lambda: ["linux", "windows", "macos"])
    dependencies: List[str] = field(default_factory=list)  # 依赖包列表

@dataclass
class ToolMetadata:
    """工具元数据"""
    # 基本信息
    name: str                              # 工具名称（唯一标识）
    display_name: str                      # 显示名称
    description: str                       # 工具描述
    version: str = "1.0.0"                 # 版本号
    author: str = ""                       # 作者
    created_at: datetime = field(default_factory=datetime.now)
    updated_at: datetime = field(default_factory=datetime.now)
    
    # 分类信息
    type: ToolType = ToolType.CUSTOM       # 工具类型
    category: str = "general"              # 工具类别
    tags: List[str] = field(default_factory=list)  # 标签
    
    # 能力描述
    capabilities: List[ToolCapability] = field(default_factory=list)
    permissions: List[ToolPermission] = field(default_factory=list)
    
    # 技术规格
    requirements: ToolRequirements = field(default_factory=ToolRequirements)
    input_schema: Dict[str, Any] = field(default_factory=dict)  # 输入Schema
    output_schema: Dict[str, Any] = field(default_factory=dict)  # 输出Schema
    examples: List[Dict[str, Any]] = field(default_factory=list)  # 使用示例
    
    # 依赖关系
    dependencies: List[str] = field(default_factory=list)  # 依赖的工具
    conflicts: List[str] = field(default_factory=list)     # 冲突的工具
    
    # 配置信息
    config_schema: Dict[str, Any] = field(default_factory=dict)  # 配置Schema
    default_config: Dict[str, Any] = field(default_factory=dict)  # 默认配置
    
    # 状态信息
    enabled: bool = True                    # 是否启用
    deprecated: bool = False                # 是否已弃用
    replacement: Optional[str] = None       # 替代工具
    
    def validate(self) -> List[str]:
        """验证工具元数据"""
        errors = []
        
        # 名称验证
        if not self.name or not self.name.strip():
            errors.append("工具名称不能为空")
        if not self.name.isidentifier():
            errors.append(f"工具名称必须是有效的标识符: {self.name}")
        
        # 版本验证
        import re
        version_pattern = r'^\d+\.\d+\.\d+$'
        if not re.match(version_pattern, self.version):
            errors.append(f"版本号格式错误: {self.version}")
        
        # 能力验证
        if not self.capabilities:
            errors.append("工具必须至少有一种能力")
        
        # 权限验证
        for perm in self.permissions:
            if perm not in ToolPermission.__members__.values():
                errors.append(f"无效的权限: {perm}")
        
        return errors
    
    def to_dict(self) -> Dict[str, Any]:
        """转换为字典"""
        return {
            "name": self.name,
            "display_name": self.display_name,
            "description": self.description,
            "version": self.version,
            "type": self.type.value,
            "category": self.category,
            "tags": self.tags,
            "capabilities": [c.value for c in self.capabilities],
            "permissions": [p.value for p in self.permissions],
            "requirements": {
                "min_memory_mb": self.requirements.min_memory_mb,
                "max_memory_mb": self.requirements.max_memory_mb,
                "min_cpu_cores": self.requirements.min_cpu_cores,
                "max_cpu_cores": self.requirements.max_cpu_cores,
                "timeout_seconds": self.requirements.timeout_seconds,
                "requires_network": self.requires_network,
                "requires_disk": self.requires_disk,
                "requires_gpu": self.requires_gpu,
                "supported_platforms": self.requirements.supported_platforms,
                "dependencies": self.requirements.dependencies
            },
            "input_schema": self.input_schema,
            "output_schema": self.output_schema,
            "examples": self.examples,
            "dependencies": self.dependencies,
            "conflicts": self.conflicts,
            "enabled": self.enabled,
            "deprecated": self.deprecated,
            "replacement": self.replacement,
            "created_at": self.created_at.isoformat(),
            "updated_at": self.updated_at.isoformat()
        }
```

```python
from typing import Dict, Any, List, Optional
from pydantic import BaseModel, Field, validator
import yaml
from pathlib import Path

class ToolTypeConfig(BaseModel):
    """工具类型配置模型"""

    icon: str = Field("📦", max_length=10)
    color: str = Field("#607D8B", regex="^#[0-9A-Fa-f]{6}$")
    default_permissions: List[str] = Field(default_factory=list)
    default_requirements: Dict[str, Any] = Field(default_factory=dict)
    security_level: str = Field("medium", regex="^(low|medium|high|critical)$")
    requires_sandbox: bool = Field(False)
    allowed_operations: List[str] = Field(default_factory=list)
    blocked_operations: List[str] = Field(default_factory=list)

    def validate_permissions(cls, v):

        valid_permissions = {p.value for p in ToolPermission}
        for perm in v:
            if perm not in valid_permissions:
                raise ValueError(f"无效的权限: {perm}")
        return v
    
    @validator("default_requirements")
    def validate_requirements(cls, v):
        required_fields = ["min_memory_mb", "max_memory_mb", "timeout_seconds"]
        for field in required_fields:
            if field not in v:
                raise ValueError(f"缺少必需字段: {field}")
        
        # 验证内存限制
        if v["min_memory_mb"] > v["max_memory_mb"]:
            raise ValueError("最小内存不能大于最大内存")
        
        # 验证超时时间
        if v["timeout_seconds"] <= 0:
            raise ValueError("超时时间必须大于0")
        
        return v

class ToolTypeManager:
    """工具类型管理器"""
    
    def __init__(self, config_path: Optional[str] = None):
        self.config_path = config_path or "config/tool_types.yaml"
        self._types: Dict[str, ToolTypeConfig] = {}
        self._load_config()
    
    def _load_config(self) -> None:
        """加载配置"""
        config_file = Path(self.config_path)
        if not config_file.exists():
            raise FileNotFoundError(f"工具类型配置文件不存在: {self.config_path}")
        
        with open(config_file, 'r', encoding='utf-8') as f:
            config_data = yaml.safe_load(f)
        
        if "tool_types" not in config_data:
            raise ValueError("配置文件缺少'tool_types'节")
        
        # 验证并加载所有工具类型
        for type_name, type_config in config_data["tool_types"].items():
            try:
                validated_config = ToolTypeConfig(**type_config)
                self._types[type_name] = validated_config
            except Exception as e:
                raise ValueError(f"工具类型'{type_name}'配置验证失败: {e}")
    
    def get_type(self, type_name: str) -> Optional[ToolTypeConfig]:
        """获取工具类型配置"""
        return self._types.get(type_name)
    
    def list_types(self) -> List[str]:
        """列出所有工具类型"""
        return list(self._types.keys())
    
    def validate_tool(self, tool_metadata: Dict[str, Any]) -> List[str]:
        """验证工具是否符合其类型要求"""
        errors = []
        
        tool_type = tool_metadata.get("type")
        if not tool_type:
            errors.append("工具缺少类型定义")
            return errors
        
        type_config = self.get_type(tool_type)
        if not type_config:
            errors.append(f"未知的工具类型: {tool_type}")
            return errors
        
        # 验证权限
        tool_permissions = set(tool_metadata.get("permissions", []))
        required_permissions = set(type_config.default_permissions)
        
        missing_permissions = required_permissions - tool_permissions
        if missing_permissions:
            errors.append(f"工具缺少必需权限: {missing_permissions}")
        
        # 验证操作
        tool_operations = set(tool_metadata.get("operations", []))
        blocked_operations = set(type_config.blocked_operations)
        
        blocked_found = tool_operations.intersection(blocked_operations)
        if blocked_found:
            errors.append(f"工具包含被禁止的操作: {blocked_found}")
        
        # 验证资源要求
        tool_requirements = tool_metadata.get("requirements", {})
        type_requirements = type_config.default_requirements
        
        # 检查内存限制
        tool_memory = tool_requirements.get("max_memory_mb", 0)
        type_max_memory = type_requirements.get("max_memory_mb", 1024)
        
        if tool_memory > type_max_memory:
            errors.append(f"工具内存需求({tool_memory}MB)超过类型限制({type_max_memory}MB)")
        
        # 检查超时时间
        tool_timeout = tool_requirements.get("timeout_seconds", 0)
        type_timeout = type_requirements.get("timeout_seconds", 30)
        
        if tool_timeout > type_timeout:
            errors.append(f"工具超时时间({tool_timeout}s)超过类型限制({type_timeout}s)")
        
        return errors
    
    def reload(self) -> None:
        """重新加载配置"""
        self._load_config()
```

```python
# backend/app/services/tools/registry.py
import json
import hashlib
from typing import Dict, List, Optional, Set, Any, Union
from dataclasses import asdict
from datetime import datetime
import threading
import asyncio
from .types import ToolMetadata, ToolType, ToolCapability, ToolPermission
from .validator import ToolTypeManager

class ToolRegistry:
    """工具注册表 - 支持多类型工具的统一管理"""
    
    def __init__(self, type_manager: Optional[ToolTypeManager] = None):
        self.type_manager = type_manager or ToolTypeManager()
        self._tools: Dict[str, ToolMetadata] = {}
        self._categories: Dict[str, Set[str]] = {}
        self._types: Dict[ToolType, Set[str]] = {}
        self._capabilities: Dict[ToolCapability, Set[str]] = {}
        self._tags: Dict[str, Set[str]] = {}
        self._lock = threading.RLock()
        self._change_listeners: List[callable] = []
        
        # 索引
        self._name_index: Dict[str, str] = {}  # 名称 -> ID
        self._display_name_index: Dict[str, str] = {}  # 显示名称 -> ID
        
        # 统计信息
        self._stats = {
            "total_tools": 0,
            "by_type": {},
            "by_category": {},
            "by_capability": {},
            "last_updated": None
        }
    
    def register(self, tool: ToolMetadata) -> str:
        """注册工具"""
        with self._lock:
            # 验证工具
            errors = tool.validate()
            if errors:
                raise ValueError(f"工具验证失败: {', '.join(errors)}")
            
            # 验证工具类型
            if self.type_manager:
                type_errors = self.type_manager.validate_tool(asdict(tool))
                if type_errors:
                    raise ValueError(f"工具类型验证失败: {', '.join(type_errors)}")
            
            # 检查名称冲突
            if tool.name in self._name_index:
                existing_id = self._name_index[tool.name]
                existing_tool = self._tools[existing_id]
                if existing_tool.version == tool.version:
                    raise ValueError(f"工具'{tool.name}'版本'{tool.version}'已存在")
                # 不同版本可以共存
                tool_id = f"{tool.name}@{tool.version}"
            else:
                tool_id = tool.name
            
            # 检查显示名称冲突
            if tool.display_name in self._display_name_index:
                raise ValueError(f"工具显示名称'{tool.display_name}'已存在")
            
            # 注册工具
            self._tools[tool_id] = tool
            self._name_index[tool.name] = tool_id
            self._display_name_index[tool.display_name] = tool_id
            
            # 更新索引
            self._update_indexes(tool_id, tool)
            
            # 更新统计
            self._update_stats()
            
            # 通知监听器
            self._notify_change("register", tool_id, tool)
            
            return tool_id
    
    def _update_indexes(self, tool_id: str, tool: ToolMetadata) -> None:
        """更新所有索引"""
        # 分类索引
        if tool.category not in self._categories:
            self._categories[tool.category] = set()
        self._categories[tool.category].add(tool_id)
        
        # 类型索引
        if tool.type not in self._types:
            self._types[tool.type] = set()
        self._types[tool.type].add(tool_id)
        
        # 能力索引
        for capability in tool.capabilities:
            if capability not in self._capabilities:
                self._capabilities[capability] = set()
            self._capabilities[capability].add(tool_id)
        
        # 标签索引
        for tag in tool.tags:
            if tag not in self._tags:
                self._tags[tag] = set()
            self._tags[tag].add(tool_id)
    
    def _update_stats(self) -> None:
        """更新统计信息"""
        self._stats["total_tools"] = len(self._tools)
        
        # 按类型统计
        self._stats["by_type"] = {
            tool_type.value: len(tool_ids)
            for tool_type, tool_ids in self._types.items()
        }
        
        # 按分类统计
        self._stats["by_category"] = {
            category: len(tool_ids)
            for category, tool_ids in self._categories.items()
        }
        
        # 按能力统计
        self._stats["by_capability"] = {
            capability.value: len(tool_ids)
            for capability, tool_ids in self._capabilities.items()
        }
        
        self._stats["last_updated"] = datetime.now().isoformat()
    
    def get_tool(self, tool_id: str) -> Optional[ToolMetadata]:
        """获取工具"""
        with self._lock:
            return self._tools.get(tool_id)
    
    def get_tool_by_name(self, name: str, version: Optional[str] = None) -> Optional[ToolMetadata]:
        """按名称获取工具"""
        with self._lock:
            if version:
                tool_id = f"{name}@{version}"
                return self._tools.get(tool_id)
            else:
                # 返回最新版本
                tool_id = self._name_index.get(name)
                if tool_id:
                    return self._tools.get(tool_id)
            return None
    
    def list_tools(
        self,
        type_filter: Optional[ToolType] = None,
        category_filter: Optional[str] = None,
        capability_filter: Optional[ToolCapability] = None,
        tag_filter: Optional[str] = None,
        enabled_only: bool = True,
        include_metadata: bool = False
    ) -> List[Union[str, ToolMetadata]]:
        """列出工具（支持多种过滤条件）"""
        with self._lock:
            # 初始工具ID集合
            if type_filter:
                tool_ids = self._types.get(type_filter, set()).copy()
            elif category_filter:
                tool_ids = self._categories.get(category_filter, set()).copy()
            elif capability_filter:
                tool_ids = self._capabilities.get(capability_filter, set()).copy()
            elif tag_filter:
                tool_ids = self._tags.get(tag_filter, set()).copy()
            else:
                tool_ids = set(self._tools.keys())
            
            # 应用其他过滤器
            filtered_ids = set()
            for tool_id in tool_ids:
                tool = self._tools[tool_id]
                
                # 启用状态过滤
                if enabled_only and not tool.enabled:
                    continue
                
                # 弃用状态过滤
                if tool.deprecated:
                    continue
                
                filtered_ids.add(tool_id)
            
            # 返回结果
            if include_metadata:
                return [self._tools[tool_id] for tool_id in filtered_ids]
            else:
                return list(filtered_ids)
    
    def search_tools(
        self,
        query: str,
        search_fields: List[str] = None,
        limit: int = 50,
        offset: int = 0
    ) -> List[ToolMetadata]:
        """搜索工具"""
        if not query or not query.strip():
            return []
        
        search_fields = search_fields or ["name", "display_name", "description", "tags"]
        query = query.lower().strip()
        
        results = []
        with self._lock:
            for tool_id, tool in self._tools.items():
                if tool.deprecated or not tool.enabled:
                    continue
                
                score = 0
                
                # 名称匹配（最高权重）
                if "name" in search_fields and query in tool.name.lower():
                    score += 10
                
                # 显示名称匹配
                if "display_name" in search_fields and query in tool.display_name.lower():
                    score += 8
                
                # 描述匹配
                if "description" in search_fields and query in tool.description.lower():
                    score += 5
                
                # 标签匹配
                if "tags" in search_fields:
                    for tag in tool.tags:
                        if query in tag.lower():
                            score += 3
                            break
                
                if score > 0:
                    results.append((score, tool))
        
        # 按分数排序
        results.sort(key=lambda x: x[0], reverse=True)
        
        # 分页
        start = offset
        end = offset + limit
        return [tool for _, tool in results[start:end]]
    
    def unregister(self, tool_id: str) -> bool:
        """注销工具"""
        with self._lock:
            if tool_id not in self._tools:
                return False
            
            tool = self._tools[tool_id]
            
            # 从索引中移除
            self._remove_from_indexes(tool_id, tool)
            
            # 从工具字典中移除
            del self._tools[tool_id]
            
            # 从名称索引中移除
            if tool.name in self._name_index and self._name_index[tool.name] == tool_id:
                del self._name_index[tool.name]
            
            # 从显示名称索引中移除
            if tool.display_name in self._display_name_index:
                del self._display_name_index[tool.display_name]
            
            # 更新统计
            self._update_stats()
            
            # 通知监听器
            self._notify_change("unregister", tool_id, tool)
            
            return True
    
    def _remove_from_indexes(self, tool_id: str, tool: ToolMetadata) -> None:
        """从所有索引中移除工具"""
        # 分类索引
        if tool.category in self._categories:
            self._categories[tool.category].discard(tool_id)
            if not self._categories[tool.category]:
                del self._categories[tool.category]
        
        # 类型索引
        if tool.type in self._types:
            self._types[tool.type].discard(tool_id)
            if not self._types[tool.type]:
                del self._types[tool.type]
        
        # 能力索引
        for capability in tool.capabilities:
            if capability in self._capabilities:
                self._capabilities[capability].discard(tool_id)
                if not self._capabilities[capability]:
                    del self._capabilities[capability]
        
        # 标签索引
        for tag in tool.tags:
            if tag in self._tags:
                self._tags[tag].discard(tool_id)
                if not self._tags[tag]:
                    del self._tags[tag]
    
    def add_change_listener(self, listener: callable) -> None:
        """添加变更监听器"""
        with self._lock:
            self._change_listeners.append(listener)
    
    def remove_change_listener(self, listener: callable) -> None:
        """移除变更监听器"""
        with self._lock:
            if listener in self._change_listeners:
                self._change_listeners.remove(listener)
    
    def _notify_change(self, action: str, tool_id: str, tool: ToolMetadata) -> None:
        """通知变更监听器"""
        event = {
            "action": action,
            "tool_id": tool_id,
            "tool": asdict(tool),
            "timestamp": datetime.now().isoformat()
        }
        
        for listener in self._change_listeners:
            try:
                listener(event)
            except Exception as e:
                # 记录错误但不中断
                print(f"变更监听器错误: {e}")
    
    def get_stats(self) -> Dict[str, Any]:
        """获取统计信息"""
        with self._lock:
            return self._stats.copy()
    
    def export(self, format: str = "json") -> str:
        """导出所有工具"""
        with self._lock:
            data = {
                "metadata": {
                    "export_time": datetime.now().isoformat(),
                    "total_tools": len(self._tools),
                    "version": "1.0"
                },
                "tools": [asdict(tool) for tool in self._tools.values()]
            }
            
            if format == "json":
                return json.dumps(data, ensure_ascii=False, indent=2)
            elif format == "yaml":
                import yaml
                return yaml.dump(data, allow_unicode=True, sort_keys=False)
            else:
                raise ValueError(f"不支持的格式: {format}")
    
    def import_tools(self, data: Union[str, Dict], merge: bool = True) -> Dict[str, List[str]]:
        """导入工具"""
        if isinstance(data, str):
            try:
                data = json.loads(data)
            except json.JSONDecodeError:
                import yaml
                data = yaml.safe_load(data)
        
        if "tools" not in data:
            raise ValueError("导入数据缺少'tools'字段")
        
        results = {
            "success": [],
            "failed": [],
            "skipped": []
        }
        
        with self._lock:
            for tool_data in data["tools"]:
                try:
                    # 创建工具元数据
                    tool = ToolMetadata(**tool_data)
                    
                    # 检查是否已存在
                    tool_id = f"{tool.name}@{tool.version}"
                    if tool_id in self._tools:
                        if merge:
                            # 合并模式：更新现有工具
                            existing_tool = self._tools[tool_id]
                            # 保留某些字段
                            tool.created_at = existing_tool.created_at
                            tool.updated_at = datetime.now()
                            results["success"].append(f"更新工具: {tool.name}@{tool.version}")

                    self.register(tool)

                    tool_version = tool_data.get("version", "unknown")
                    results["failed"].append(f"{tool_name}@{tool_version}: {str(e)}")
```

```python
# backend/app/services/tools/decorators.py
import functools
import inspect
from typing import Dict, Any, Optional, Callable, List
from datetime import datetime
from .types import ToolMetadata, ToolType, ToolCapability, ToolPermission, ToolRequirements
from .registry import ToolRegistry

# 全局工具注册表实例
_tool_registry = ToolRegistry()

def tool(
    name: Optional[str] = None,
    display_name: Optional[str] = None,
    description: str = "",
    tool_type: ToolType = ToolType.CUSTOM,
    category: str = "general",
    version: str = "1.0.0",
    author: str = "",
    capabilities: Optional[List[ToolCapability]] = None,
    permissions: Optional[List[ToolPermission]] = None,
    requirements: Optional[ToolRequirements] = None,
    input_schema: Optional[Dict[str, Any]] = None,
    output_schema: Optional[Dict[str, Any]] = None,
    examples: Optional[List[Dict[str, Any]]] = None,
    dependencies: Optional[List[str]] = None,
    tags: Optional[List[str]] = None,
    enabled: bool = True
):
    """
    工具装饰器 - 用于注册工具函数
    
    用法:
    @tool(
        name="read_file",
        display_name="读取文件",
        description="读取文件内容",
        tool_type=ToolType.FILE_OPERATION,
        capabilities=[ToolCapability.READ],
        permissions=[ToolPermission.READ_FILE]
    )
    async def read_file(file_path: str, encoding: str = "utf-8") -> str:
        # 工具实现
        pass
    """
    def decorator(func: Callable):
        # 获取工具名称
        tool_name = name or func.__name__
        
        # 获取显示名称
        tool_display_name = display_name or tool_name.replace("_", " ").title()
        
        # 获取描述
        tool_description = description or (func.__doc__ or "").strip().split("\n")[0]
        
        # 从函数签名生成输入Schema
        if input_schema is None:
            input_schema = generate_schema_from_function(func)
        
        # 创建工具元数据
        metadata = ToolMetadata(
            name=tool_name,
            display_name=tool_display_name,
            description=tool_description,
            type=tool_type,
            category=category,
            version=version,
            author=author,
            capabilities=capabilities or [],
            permissions=permissions or [],
            requirements=requirements or ToolRequirements(),
            input_schema=input_schema,
            output_schema=output_schema or {},
            examples=examples or [],
            dependencies=dependencies or [],
            tags=tags or [],
            enabled=enabled,
            created_at=datetime.now(),
            updated_at=datetime.now()
        )
        
        # 注册工具
        try:
            tool_id = _tool_registry.register(metadata)
            
            # 将工具ID附加到函数
            func.__tool_id__ = tool_id
            func.__tool_metadata__ = metadata
            
            # 添加工具执行包装器
            @functools.wraps(func)
            async def wrapper(*args, **kwargs):
                # 执行前的钩子
                start_time = datetime.now()
                
                try:
                    # 执行原始函数
                    result = await func(*args, **kwargs)
                    
                    # 执行后的钩子
                    end_time = datetime.now()
                    execution_time = (end_time - start_time).total_seconds()
                    
                    # 记录执行指标
                    record_tool_execution(
                        tool_id=tool_id,
                        success=True,
                        execution_time=execution_time,
                        args=args,
                        kwargs=kwargs
                    )
                    
                    return result
                    
                except Exception as e:
                    # 错误处理
                    end_time = datetime.now()
                    execution_time = (end_time - start_time).total_seconds()
                    
                    # 记录错误指标
                    record_tool_execution(
                        tool_id=tool_id,
                        success=False,
                        execution_time=execution_time,
                        error=str(e),
                        args=args,
                        kwargs=kwargs
                    )
                    
                    # 重新抛出异常
                    raise
            
            # 将元数据附加到包装器
            wrapper.__tool_id__ = tool_id
            wrapper.__tool_metadata__ = metadata
            
            return wrapper
            
        except Exception as e:
            raise ValueError(f"注册工具'{tool_name}'失败: {e}")
    
    return decorator

def generate_schema_from_function(func: Callable) -> Dict[str, Any]:
    """从函数签名生成JSON Schema"""
    import inspect
    from typing import get_type_hints
    
    schema = {
        "type": "object",
        "properties": {},
        "required": []
    }
    
    # 获取函数签名
    sig = inspect.signature(func)
    type_hints = get_type_hints(func)
    
    for param_name, param in sig.parameters.items():
        # 跳过self参数
        if param_name == "self":
            continue
        
        # 获取参数类型
        param_type = type_hints.get(param_name, str)
        type_name = get_type_name(param_type)
        
        # 创建参数Schema
        param_schema = {
            "type": type_name,
            "description": f"参数: {param_name}"
        }
        
        # 添加默认值
        if param.default != inspect.Parameter.empty:
            param_schema["default"] = param.default
        else:
            schema["required"].append(param_name)
        
        # 添加枚举值（如果有）
        if hasattr(param_type, "__args__") and hasattr(param_type, "__origin__"):
            # 处理Union类型
            pass
        elif hasattr(param_type, "__members__"):
            # 处理枚举类型
            param_schema["enum"] = list(param_type.__members__.keys())
        
        schema["properties"][param_name] = param_schema
    
    return schema

def get_type_name(python_type) -> str:
    """将Python类型转换为JSON Schema类型"""
    type_mapping = {
        str: "string",
        int: "integer",
        float: "number",
        bool: "boolean",
        list: "array",
        dict: "object",
        type(None): "null"
    }
    
    # 检查直接映射
    if python_type in type_mapping:
        return type_mapping[python_type]
    
    # 检查是否是Optional类型
    import typing
    if hasattr(python_type, "__origin__") and python_type.__origin__ is typing.Union:
        # 提取非None类型
        non_none_types = [t for t in python_type.__args__ if t is not type(None)]
        if non_none_types:
            return get_type_name(non_none_types[0])
    
    # 默认返回string
    return "string"
```

```python
# backend/app/services/tools/loader.py
import importlib
import pkgutil
import sys
from pathlib import Path
from typing import List, Dict, Any, Optional, Set
import yaml
from .registry import ToolRegistry, ToolMetadata
from .types import ToolType, ToolCapability, ToolPermission, ToolRequirements

class ToolLoader:
    """工具加载器 - 支持动态发现和加载工具"""
    
    def __init__(self, registry: ToolRegistry, config_path: Optional[str] = None):
        self.registry = registry
        self.config_path = config_path or "config/tools_config.yaml"
        self.loaded_modules: Set[str] = set()
        self.failed_modules: Dict[str, str] = {}
        self._load_config()
    
    def _load_config(self) -> None:
        """加载配置"""
        config_file = Path(self.config_path)
        if not config_file.exists():
            self.config = {
                "auto_discover": True,
                "scan_paths": ["app/services/tools"],
                "exclude_patterns": ["**/test_*.py", "**/*_test.py", "**/__pycache__/**"],
                "preload_tools": [],
                "disabled_tools": []
            }
            return
        
        with open(config_file, 'r', encoding='utf-8') as f:
            self.config = yaml.safe_load(f) or {}
    
    def discover_tool_modules(self) -> List[str]:
        """发现所有工具模块"""
        if not self.config.get("auto_discover", True):
            return []
        
        tool_modules = []
        scan_paths = self.config.get("scan_paths", ["app/services/tools"])
        exclude_patterns = self.config.get("exclude_patterns", [])
        
        for scan_path in scan_paths:
            scan_dir = Path(scan_path)
            if not scan_dir.exists():
                continue
            
            # 递归查找Python文件
            for py_file in scan_dir.rglob("*.py"):
                # 检查是否在排除模式中
                file_path = str(py_file)
                if any(self._match_pattern(file_path, pattern) for pattern in exclude_patterns):
                    continue
                
                # 转换为模块路径
                try:
                    module_path = self._file_to_module_path(py_file, scan_dir)
                    if module_path and module_path not in self.loaded_modules:
                        tool_modules.append(module_path)
                except ValueError as e:
                    print(f"无法转换文件路径: {py_file}, 错误: {e}")
        
        return tool_modules
    
    def _match_pattern(self, file_path: str, pattern: str) -> bool:
        """检查文件路径是否匹配模式"""
        from fnmatch import fnmatch
        
        # 将**/*.py模式转换为fnmatch可识别的模式
        if pattern.startswith("**/"):
            # **/test_*.py -> */test_*.py
            pattern = pattern[3:]
        
        return fnmatch(file_path, pattern)
    
    def _file_to_module_path(self, file_path: Path, base_dir: Path) -> str:
        """将文件路径转换为模块路径"""
        # 计算相对路径
        try:
            rel_path = file_path.relative_to(base_dir)
        except ValueError:
            # 文件不在基础目录下
            raise ValueError(f"文件不在基础目录下: {file_path}")
        
        # 转换为模块路径
        module_parts = []
        for part in rel_path.parts:
            if part.endswith(".py"):
                module_parts.append(part[:-3])  # 移除.py
            else:
                module_parts.append(part)
        
        # 构建完整模块路径
        base_module = str(base_dir).replace("/", ".").replace("\\", ".")
        module_path = f"{base_module}.{'.'.join(module_parts)}"
        
        return module_path
    
    def load_tool_module(self, module_path: str) -> Dict[str, Any]:
        """加载工具模块"""
        result = {
            "module": module_path,
            "loaded": False,
            "tools": [],
            "errors": []
        }
        
        try:
            # 导入模块
            module = importlib.import_module(module_path)
            
            # 查找模块中的所有工具函数
            tools_found = []
            for attr_name in dir(module):
                attr = getattr(module, attr_name)
                
                # 检查是否被@tool装饰
                if hasattr(attr, '__tool_metadata__'):
                    metadata = getattr(attr, '__tool_metadata__')
                    tools_found.append({
                        "name": metadata.name,
                        "display_name": metadata.display_name,
                        "type": metadata.type.value,
                        "function": attr_name
                    })
            
            if tools_found:
                result["loaded"] = True
                result["tools"] = tools_found
                self.loaded_modules.add(module_path)
                print(f"成功加载模块: {module_path}, 找到 {len(tools_found)} 个工具")
            else:
                result["errors"].append(f"模块中未找到工具: {module_path}")
                
        except ImportError as e:
            result["errors"].append(f"导入模块失败: {e}")
            self.failed_modules[module_path] = str(e)
        except Exception as e:
            result["errors"].append(f"加载模块时出错: {e}")
            self.failed_modules[module_path] = str(e)
        
        return result
    
    def load_all_tools(self) -> Dict[str, List[str]]:
        """加载所有工具"""
        results = {
            "loaded": [],
            "failed": [],
            "skipped": []
        }
        
        # 预加载工具
        preload_tools = self.config.get("preload_tools", [])
        for tool_config in preload_tools:
            try:
                self._load_predefined_tool(tool_config)
                results["loaded"].append(f"预加载工具: {tool_config.get('name', 'unknown')}")
            except Exception as e:
                results["failed"].append(f"预加载工具失败: {tool_config.get('name', 'unknown')}: {e}")
        
        # 自动发现和加载
        if self.config.get("auto_discover", True):
            tool_modules = self.discover_tool_modules()
            
            for module_path in tool_modules:
                if module_path in self.loaded_modules:
                    results["skipped"].append(f"模块已加载: {module_path}")
                    continue
                
                load_result = self.load_tool_module(module_path)
                if load_result["loaded"]:
                    results["loaded"].append(f"模块: {module_path}")
                    results["loaded"].extend([
                        f"  - {tool['name']} ({tool['type']})"
                        for tool in load_result["tools"]
                    ])
                else:
                    results["failed"].append(f"模块: {module_path}")
                    results["failed"].extend([
                        f"  - 错误: {error}"
                        for error in load_result["errors"]
                    ])
        
        return results
    
    def _load_predefined_tool(self, config: Dict[str, Any]) -> None:
        """加载预定义工具"""
        # 创建工具元数据
        metadata = ToolMetadata(
            name=config["name"],
            display_name=config.get("display_name", config["name"]),
            description=config.get("description", ""),
            type=ToolType(config.get("type", "custom")),
            category=config.get("category", "general"),
            version=config.get("version", "1.0.0"),
            author=config.get("author", ""),
            capabilities=[ToolCapability(c) for c in config.get("capabilities", [])],
            permissions=[ToolPermission(p) for p in config.get("permissions", [])],
            requirements=ToolRequirements(**config.get("requirements", {})),
            input_schema=config.get("input_schema", {}),
            output_schema=config.get("output_schema", {}),
            examples=config.get("examples", []),
            dependencies=config.get("dependencies", []),
            tags=config.get("tags", []),
            enabled=config.get("enabled", True)
        )
        
        # 创建工具函数
        def tool_function(**kwargs):
            # 这里可以调用实际的工具实现
            # 对于预定义工具，可能需要从配置中加载实现
            implementation = config.get("implementation")
            if implementation:
                # 动态加载实现
                module_name, func_name = implementation.rsplit(".", 1)
                module = importlib.import_module(module_name)
                func = getattr(module, func_name)
                return func(**kwargs)
            else:
                # 返回模拟结果
                return {
                    "status": "success",
                    "message": f"预定义工具 '{config['name']}' 执行成功",
                    "data": kwargs
                }
        
        # 添加元数据到函数
        tool_function.__tool_metadata__ = metadata
        
        # 注册工具
        self.registry.register(metadata)
    
    def reload(self) -> Dict[str, List[str]]:
        """重新加载所有工具"""
        # 清空已加载模块
        self.loaded_modules.clear()
        self.failed_modules.clear()
        
        # 重新加载配置
        self._load_config()
        
        # 重新加载所有工具
        return self.load_all_tools()
    
    def get_loaded_modules(self) -> List[str]:
        """获取已加载的模块"""
        return list(self.loaded_modules)
    
    def get_failed_modules(self) -> Dict[str, str]:
        """获取加载失败的模块"""
        return self.failed_modules.copy()
```

```python
import asyncio
import json
from typing import Dict, List, Any, Optional, Set, Union

from dataclasses import asdict

class ToolManager:
    """工具管理器 - 统一管理工具的生命周期"""
    
    def __init__(
        self,
        registry: Optional[ToolRegistry] = None,
        loader: Optional[ToolLoader] = None,
        config_path: Optional[str] = None
    ):
        self.registry = registry or ToolRegistry()
        self.loader = loader or ToolLoader(self.registry, config_path)
        self._initialized = False
        self._lock = threading.RLock()
        self._health_check_interval = 60  # 健康检查间隔（秒）
        self._last_health_check = None
        
        # 工具健康状态
        self._tool_health: Dict[str, Dict[str, Any]] = {}
        
        # 工具使用统计
        self._tool_stats: Dict[str, Dict[str, Any]] = {}
        
        # 工具依赖图
        self._dependency_graph: Dict[str, Set[str]] = {}
        
        # 事件监听器
        self._event_listeners: Dict[str, List[callable]] = {
            "tool_registered": [],
            "tool_unregistered": [],
            "tool_enabled": [],
            "tool_disabled": [],
            "tool_executed": [],
            "tool_failed": []
        }
    
    async def initialize(self) -> None:
        """初始化工具管理器"""
        with self._lock:
            if self._initialized:
                return
            
            print("正在初始化工具管理器...")
            
            # 加载所有工具
            load_results = self.loader.load_all_tools()
            
            # 构建依赖图
            self._build_dependency_graph()
            
            # 初始化健康状态
            self._initialize_health_status()
            
            # 初始化使用统计
            self._initialize_stats()
            
            # 启动健康检查任务
            asyncio.create_task(self._health_check_task())
            
            self._initialized = True
            
            print(f"工具管理器初始化完成。加载结果: {json.dumps(load_results, ensure_ascii=False, indent=2)}")
    
    def _build_dependency_graph(self) -> None:
        """构建工具依赖图"""
        self._dependency_graph.clear()
        
        # 添加所有工具节点
        for tool_id, tool in self.registry._tools.items():
            self._dependency_graph[tool_id] = set(tool.dependencies)
    
    def _initialize_health_status(self) -> None:
        """初始化工具健康状态"""
        for tool_id in self.registry._tools.keys():
            self._tool_health[tool_id] = {
                "status": "healthy",
                "last_check": datetime.now().isoformat(),
                "error_count": 0,
                "consecutive_errors": 0,
                "last_error": None,
                "last_success": None
            }
    
    def _initialize_stats(self) -> None:
        """初始化工具使用统计"""
        for tool_id in self.registry._tools.keys():
            self._tool_stats[tool_id] = {
                "total_executions": 0,
                "successful_executions": 0,
                "failed_executions": 0,
                "total_execution_time": 0.0,
                "avg_execution_time": 0.0,
                "last_execution": None,
                "concurrent_executions": 0,
                "max_concurrent_executions": 0
            }
    
    async def _health_check_task(self) -> None:
        """健康检查任务"""
        while True:
            try:
                await self._perform_health_check()
            except Exception as e:
                print(f"健康检查任务出错: {e}")
            
            await asyncio.sleep(self._health_check_interval)
    
    async def _perform_health_check(self) -> None:
        """执行健康检查"""
        with self._lock:
            self._last_health_check = datetime.now()
            
            for tool_id, tool in self.registry._tools.items():
                if not tool.enabled:
                    continue
                
                try:
                    # 检查工具依赖是否可用
                    missing_deps = self._check_dependencies(tool_id)
                    if missing_deps:
                        self._update_health_status(
                            tool_id,
                            "unhealthy",
                            f"缺少依赖: {', '.join(missing_deps)}"
                        )
                        continue
                    
                    # 检查工具资源要求
                    if not self._check_resource_requirements(tool):
                        self._update_health_status(
                            tool_id,
                            "unhealthy",
                            "资源要求不满足"
                        )
                        continue
                    
                    # 执行健康检查（如果有）
                    if hasattr(tool, 'health_check'):
                        health_result = await tool.health_check()
                        if not health_result.get("healthy", True):
                            self._update_health_status(
                                tool_id,
                                "unhealthy",
                                health_result.get("message", "健康检查失败")
                            )
                            continue
                    
                    # 工具健康
                    self._update_health_status(tool_id, "healthy")
                    
                except Exception as e:
                    self._update_health_status(
                        tool_id,
                        "unhealthy",
                        f"健康检查异常: {str(e)}"
                    )
    
    def _check_dependencies(self, tool_id: str) -> List[str]:
        """检查工具依赖"""
        missing = []
        dependencies = self._dependency_graph.get(tool_id, set())
        
        for dep_id in dependencies:
            dep_tool = self.registry.get_tool(dep_id)
            if not dep_tool or not dep_tool.enabled:
                missing.append(dep_id)
        
        return missing
    
    def _check_resource_requirements(self, tool: ToolMetadata) -> bool:
        """检查资源要求"""
        # 这里可以检查系统资源是否满足工具要求
        # 例如：内存、CPU、磁盘空间等
        
        # 暂时返回True，实际实现需要检查系统资源
        return True
    
    def _update_health_status(
        self,
        tool_id: str,
        status: str,
        message: Optional[str] = None
    ) -> None:
        """更新工具健康状态"""
        health = self._tool_health.get(tool_id, {})
        
        old_status = health.get("status", "unknown")
        health["status"] = status
        health["last_check"] = datetime.now().isoformat()
        
        if status == "healthy":
            health["consecutive_errors"] = 0
            health["last_success"] = datetime.now().isoformat()
        else:
            health["error_count"] = health.get("error_count", 0) + 1
            health["consecutive_errors"] = health.get("consecutive_errors", 0) + 1
            health["last_error"] = message or "未知错误"
        
        self._tool_health[tool_id] = health
        
        # 触发事件
        if old_status != status:
            self._trigger_event("tool_health_changed", {
                "tool_id": tool_id,
                "old_status": old_status,
                "new_status": status,
                "message": message,
                "timestamp": datetime.now().isoformat()
            })
    
    def record_tool_execution(
        self,
        tool_id: str,
        success: bool,
        execution_time: float,
        error: Optional[str] = None
    ) -> None:
        """记录工具执行"""
        with self._lock:
            stats = self._tool_stats.get(tool_id, {})
            
            # 更新统计
            stats["total_executions"] = stats.get("total_executions", 0) + 1
            stats["total_execution_time"] = stats.get("total_execution_time", 0.0) + execution_time
            stats["avg_execution_time"] = stats["total_execution_time"] / stats["total_executions"]
            stats["last_execution"] = datetime.now().isoformat()
            
            if success:
                stats["successful_executions"] = stats.get("successful_executions", 0) + 1
            else:
                stats["failed_executions"] = stats.get("failed_executions", 0) + 1
            
            # 更新并发执行数
            stats["concurrent_executions"] = stats.get("concurrent_executions", 0) + 1
            stats["max_concurrent_executions"] = max(
                stats.get("max_concurrent_executions", 0),
                stats["concurrent_executions"]
            )
            
            self._tool_stats[tool_id] = stats
            
            # 触发事件
            event_type = "tool_executed" if success else "tool_failed"
            self._trigger_event(event_type, {
                "tool_id": tool_id,
                "success": success,
                "execution_time": execution_time,
                "error": error,
                "timestamp": datetime.now().isoformat()
            })
    
    def finish_tool_execution(self, tool_id: str) -> None:
        """完成工具执行（减少并发计数）"""
        with self._lock:
            stats = self._tool_stats.get(tool_id, {})
            current = stats.get("concurrent_executions", 0)
            if current > 0:
                stats["concurrent_executions"] = current - 1
                self._tool_stats[tool_id] = stats
    
    def get_tool_health(self, tool_id: str) -> Dict[str, Any]:
        """获取工具健康状态"""
        return self._tool_health.get(tool_id, {}).copy()
    
    def get_tool_stats(self, tool_id: str) -> Dict[str, Any]:
        """获取工具使用统计"""
        return self._tool_stats.get(tool_id, {}).copy()
    
    def get_all_health_status(self) -> Dict[str, Dict[str, Any]]:
        """获取所有工具健康状态"""
        return {
            tool_id: health.copy()
            for tool_id, health in self._tool_health.items()
        }
    
    def get_all_stats(self) -> Dict[str, Dict[str, Any]]:
        """获取所有工具使用统计"""
        return {
            tool_id: stats.copy()
            for tool_id, stats in self._tool_stats.items()
        }
    
    def enable_tool(self, tool_id: str) -> bool:
        """启用工具"""
        with self._lock:
            tool = self.registry.get_tool(tool_id)
            if not tool:
                return False
            
            if tool.enabled:
                return True  # 已经启用
            
            tool.enabled = True
            tool.updated_at = datetime.now()
            
            # 触发事件
            self._trigger_event("tool_enabled", {
                "tool_id": tool_id,
                "timestamp": datetime.now().isoformat()
            })
            
            return True
    
    def disable_tool(self, tool_id: str, reason: Optional[str] = None) -> bool:
        """禁用工具"""
        with self._lock:
            tool = self.registry.get_tool(tool_id)
            if not tool:
                return False
            
            if not tool.enabled:
                return True  # 已经禁用
            
            tool.enabled = False
            tool.updated_at = datetime.now()
            
            # 触发事件
            self._trigger_event("tool_disabled", {
                "tool_id": tool_id,
                "reason": reason,
                "timestamp": datetime.now().isoformat()
            })
            
            return True
    
    def add_event_listener(self, event_type: str, listener: callable) -> None:
        """添加事件监听器"""
        with self._lock:
            if event_type not in self._event_listeners:
                self._event_listeners[event_type] = []
            self._event_listeners[event_type].append(listener)
    
    def remove_event_listener(self, event_type: str, listener: callable) -> None:
        """移除事件监听器"""
        with self._lock:
            if event_type in self._event_listeners and listener in self._event_listeners[event_type]:
                self._event_listeners[event_type].remove(listener)
    
    def _trigger_event(self, event_type: str, data: Dict[str, Any]) -> None:
        """触发事件"""
        listeners = self._event_listeners.get(event_type, [])
        for listener in listeners:
            try:
                listener(event_type, data)
            except Exception as e:
                print(f"事件监听器错误 ({event_type}): {e}")
    
    def get_dependency_tree(self, tool_id: str) -> Dict[str, Any]:
        """获取工具依赖树"""
        def build_tree(current_id: str, visited: Set[str] = None) -> Dict[str, Any]:
            if visited is None:
                visited = set()
            
            if current_id in visited:
                return {"id": current_id, "circular": True}
            
            visited.add(current_id)
            
            tool = self.registry.get_tool(current_id)
            if not tool:
                return {"id": current_id, "error": "工具不存在"}
            
            node = {
                "id": current_id,
                "name": tool.name,
                "display_name": tool.display_name,
                "type": tool.type.value,
                "enabled": tool.enabled,
                "dependencies": []
            }
            
            for dep_id in tool.dependencies:
                dep_node = build_tree(dep_id, visited.copy())
                node["dependencies"].append(dep_node)
            
            return node
        
        return build_tree(tool_id)
    
    def validate_tool_chain(self, tool_ids: List[str]) -> List[str]:
        """验证工具链"""
        errors = []
        visited = set()
        
        for tool_id in tool_ids:
            if tool_id in visited:
                errors.append(f"工具重复: {tool_id}")
                continue
            
            tool = self.registry.get_tool(tool_id)
            if not tool:
                errors.append(f"工具不存在: {tool_id}")
                continue
            
            if not tool.enabled:
                errors.append(f"工具未启用: {tool_id}")
                continue
            
            # 检查依赖
            missing_deps = self._check_dependencies(tool_id)
            if missing_deps:
                errors.append(f"工具 {tool_id} 缺少依赖: {', '.join(missing_deps)}")
            
            visited.add(tool_id)
        
        return errors
    
    def export_tools(
        self,
        format: str = "json",
        include_disabled: bool = False,
        include_deprecated: bool = False
    ) -> str:
        """导出工具"""
        tools_data = []
        
        for tool_id, tool in self.registry._tools.items():
            if (not tool.enabled and not include_disabled) or \
               (tool.deprecated and not include_deprecated):
                continue
            
            tool_dict = asdict(tool)
            tool_dict["id"] = tool_id
            tool_dict["health"] = self.get_tool_health(tool_id)
            tool_dict["stats"] = self.get_tool_stats(tool_id)
            
            tools_data.append(tool_dict)
        
        export_data = {
            "metadata": {
                "export_time": datetime.now().isoformat(),
                "total_tools": len(tools_data),
                "format_version": "1.0"
            },
            "tools": tools_data
        }
        
        if format == "json":
            return json.dumps(export_data, ensure_ascii=False, indent=2)
        elif format == "yaml":
            import yaml
            return yaml.dump(export_data, allow_unicode=True, sort_keys=False)
        else:
            raise ValueError(f"不支持的格式: {format}")
    
    def import_tools(self, data: Union[str, Dict], merge: bool = True) -> Dict[str, List[str]]:
        """导入工具"""
        return self.registry.import_tools(data, merge)
```

```python
def has_cycle(graph: Dict[str, Set[str]]) -> bool:
    """检查图中是否有环"""
    visited = set()
    recursion_stack = set()
    
    def dfs(node: str) -> bool:
        if node in recursion_stack:
            return True
        if node in visited:
            return False
        
        visited.add(node)
        recursion_stack.add(node)
        
        for neighbor in graph.get(node, set()):
            if dfs(neighbor):
                return True
        
        recursion_stack.remove(node)
        return False
    
    for node in graph:
        if dfs(node):
            return True
    
    return False

class ToolChainExecutor:
    """工具链执行器"""
    
    def __init__(self, tool_manager, registry):
        self.tool_manager = tool_manager
        self.registry = registry
        self.running_chains: Dict[str, asyncio.Task] = {}
        self.chain_results: Dict[str, ChainResult] = {}
        self._lock = asyncio.Lock()
    
    async def execute_chain(
        self,
        chain_def: ChainDefinition,
        inputs: Dict[str, Any],
        context: Optional[Dict[str, Any]] = None
    ) -> ChainResult:
        """执行工具链"""
        chain_id = f"{chain_def.id}_{datetime.now().strftime('%Y%m%d_%H%M%S')}"
        
        # 创建执行任务
        task = asyncio.create_task(
            self._execute_chain_internal(chain_id, chain_def, inputs, context or {})
        )
        
        # 保存运行状态
        async with self._lock:
            self.running_chains[chain_id] = task
        
        try:
            # 等待任务完成
            result = await task
            return result
        finally:
            # 清理运行状态
            async with self._lock:
                if chain_id in self.running_chains:
                    del self.running_chains[chain_id]
    
    async def _execute_chain_internal(
        self,
        chain_id: str,
        chain_def: ChainDefinition,
        inputs: Dict[str, Any],
        context: Dict[str, Any]
    ) -> ChainResult:
        """内部执行工具链"""
        start_time = datetime.now()
        
        # 初始化结果
        chain_result = ChainResult(
            chain_id=chain_id,
            status=ChainStatus.RUNNING,
            steps={},
            started_at=start_time
        )
        
        try:
            # 验证输入
            validation_errors = self._validate_inputs(inputs, chain_def.input_schema)
            if validation_errors:
                chain_result.status = ChainStatus.FAILED
                chain_result.error = f"输入验证失败: {', '.join(validation_errors)}"
                chain_result.completed_at = datetime.now()
                chain_result.total_execution_time = (
                    chain_result.completed_at - start_time
                ).total_seconds()
                return chain_result
            
            # 拓扑排序步骤
            sorted_steps = self._topological_sort(chain_def.steps)
            
            # 执行上下文
            execution_context = {
                "inputs": inputs,
                "context": context,
                "results": {},
                "variables": {}
            }
            
            # 执行步骤
            step_results = {}
            for step in sorted_steps:
                step_result = await self._execute_step(
                    step, chain_def, execution_context, chain_id
                )
                step_results[step.id] = step_result
                
                # 更新执行上下文
                execution_context["results"][step.id] = step_result
                if step_result.output is not None:
                    execution_context["variables"][step.id] = step_result.output
                
                # 检查步骤结果
                if step_result.status == StepStatus.FAILED and not step.continue_on_error:
                    chain_result.status = ChainStatus.FAILED
                    chain_result.error = f"步骤 '{step.name}' 执行失败: {step_result.error}"
                    break
            
            # 更新链结果
            chain_result.steps = step_results
            
            # 检查最终状态
            if chain_result.status == ChainStatus.RUNNING:
                # 所有步骤执行完成
                chain_result.status = ChainStatus.COMPLETED
                
                # 生成输出
                chain_result.output = self._generate_output(
                    chain_def, execution_context
                )
            
        except asyncio.CancelledError:
            chain_result.status = ChainStatus.CANCELLED
            chain_result.error = "工具链执行被取消"
        except Exception as e:
            chain_result.status = ChainStatus.FAILED
            chain_result.error = f"工具链执行异常: {str(e)}"
        finally:
            # 计算执行时间
            end_time = datetime.now()
            chain_result.completed_at = end_time
            chain_result.total_execution_time = (end_time - start_time).total_seconds()
            
            # 保存结果
            async with self._lock:
                self.chain_results[chain_id] = chain_result
        
        return chain_result
    
    def _validate_inputs(
        self,
        inputs: Dict[str, Any],
        input_schema: Dict[str, Any]
    ) -> List[str]:
        """验证输入数据"""
        errors = []
        
        # 检查必需字段
        required_fields = input_schema.get("required", [])
        for field in required_fields:
            if field not in inputs:
                errors.append(f"缺少必需字段: {field}")
        
        # 检查字段类型（简化验证）
        properties = input_schema.get("properties", {})
        for field, value in inputs.items():
            if field in properties:
                field_schema = properties[field]
                field_type = field_schema.get("type")
                
                if field_type == "string" and not isinstance(value, str):
                    errors.append(f"字段 '{field}' 应该是字符串类型")
                elif field_type == "integer" and not isinstance(value, int):
                    errors.append(f"字段 '{field}' 应该是整数类型")
                elif field_type == "number" and not isinstance(value, (int, float)):
                    errors.append(f"字段 '{field}' 应该是数字类型")
                elif field_type == "boolean" and not isinstance(value, bool):
                    errors.append(f"字段 '{field}' 应该是布尔类型")
                elif field_type == "array" and not isinstance(value, list):
                    errors.append(f"字段 '{field}' 应该是数组类型")
                elif field_type == "object" and not isinstance(value, dict):
                    errors.append(f"字段 '{field}' 应该是对象类型")
        
        return errors
    
    def _topological_sort(self, steps: List[StepDefinition]) -> List[StepDefinition]:
        """拓扑排序步骤"""
        # 构建图
        graph = {step.id: step for step in steps}
        dependencies = {step.id: set(step.depends_on) for step in steps}
        
        # Kahn算法进行拓扑排序
        in_degree = {step.id: 0 for step in steps}
        for step in steps:
            for dep in step.depends_on:
                if dep in in_degree:
                    in_degree[dep] += 1
        
        # 找到入度为0的节点
        queue = [step_id for step_id, degree in in_degree.items() if degree == 0]
        sorted_steps = []
        
        while queue:
            step_id = queue.pop(0)
            sorted_steps.append(graph[step_id])
            
            # 减少依赖节点的入度
            for step in steps:
                if step_id in step.depends_on:
                    in_degree[step.id] -= 1
                    if in_degree[step.id] == 0:
                        queue.append(step.id)
        
        # 检查是否有环
        if len(sorted_steps) != len(steps):
            raise ValueError("步骤依赖关系存在循环")
        
        return sorted_steps
    
    async def _execute_step(
        self,
        step: StepDefinition,
        chain_def: ChainDefinition,
        context: Dict[str, Any],
        chain_id: str
    ) -> StepResult:
        """执行单个步骤"""
        start_time = datetime.now()
        step_result = StepResult(
            step_id=step.id,
            status=StepStatus.PENDING,
            started_at=start_time
        )
        
        try:
            # 检查执行条件
            if step.condition:
                if not self._evaluate_condition(step.condition, context):
                    step_result.status = StepStatus.SKIPPED
                    step_result.completed_at = datetime.now()
                    step_result.execution_time = (
                        step_result.completed_at - start_time
                    ).total_seconds()
                    return step_result
            
            # 准备参数
            params = self._prepare_step_params(step.params, context)
            
            # 获取工具
            tool = self.registry.get_tool_by_name(step.tool)
            if not tool:
                step_result.status = StepStatus.FAILED
                step_result.error = f"工具不存在: {step.tool}"
                step_result.completed_at = datetime.now()
                step_result.execution_time = (
                    step_result.completed_at - start_time
                ).total_seconds()
                return step_result
            
            # 检查工具是否启用
            if not tool.enabled:
                step_result.status = StepStatus.FAILED
                step_result.error = f"工具未启用: {step.tool}"
                step_result.completed_at = datetime.now()
                step_result.execution_time = (
                    step_result.completed_at - start_time
                ).total_seconds()
                return step_result
            
            # 执行工具（支持重试）
            step_result.status = StepStatus.RUNNING
            
            max_retries = step.retry_policy.get("max_retries", 3)
            backoff_factor = step.retry_policy.get("backoff_factor", 2.0)
            retryable_errors = step.retry_policy.get("retryable_errors", [])
            timeout = step.timeout or tool.requirements.timeout_seconds
            
            for attempt in range(max_retries + 1):
                try:
                    # 获取工具函数
                    tool_func = self._get_tool_function(step.tool)
                    if not tool_func:
                        raise ValueError(f"无法获取工具函数: {step.tool}")
                    
                    # 设置超时
                    execution_task = asyncio.wait_for(
                        tool_func(**params),
                        timeout=timeout
                    )
                    
                    # 执行工具
                    output = await execution_task
                    
                    # 记录成功
                    step_result.status = StepStatus.SUCCESS
                    step_result.output = output
                    break
                    
                except asyncio.TimeoutError:
                    if attempt == max_retries:
                        step_result.status = StepStatus.FAILED
                        step_result.error = f"执行超时 ({timeout}秒)"
                    else:
                        # 重试
                        wait_time = backoff_factor ** attempt
                        await asyncio.sleep(wait_time)
                        continue
                        
                except Exception as e:
                    error_msg = str(e)
                    
                    # 检查是否可重试的错误
                    is_retryable = any(
                        retryable_error in error_msg.lower()
                        for retryable_error in retryable_errors
                    )
                    
                    if not is_retryable or attempt == max_retries:
                        step_result.status = StepStatus.FAILED
                        step_result.error = error_msg
                        break
                    else:
                        # 重试
                        wait_time = backoff_factor ** attempt
                        await asyncio.sleep(wait_time)
                        continue
            
            # 记录执行时间
            step_result.completed_at = datetime.now()
            step_result.execution_time = (
                step_result.completed_at - start_time
            ).total_seconds()
            
            # 记录工具执行统计
            self.tool_manager.record_tool_execution(
                tool_id=step.tool,
                success=step_result.status == StepStatus.SUCCESS,
                execution_time=step_result.execution_time,
                error=step_result.error
            )
            
        except Exception as e:
            step_result.status = StepStatus.FAILED
            step_result.error = f"步骤执行异常: {str(e)}"
            step_result.completed_at = datetime.now()
            step_result.execution_time = (
                step_result.completed_at - start_time
            ).total_seconds()
        
        return step_result
    
    def _evaluate_condition(self, condition: str, context: Dict[str, Any]) -> bool:
        """评估执行条件"""
        try:
            # 安全地评估Python表达式
            # 注意：实际实现中应该使用更安全的表达式评估库
            # 这里使用简单的变量替换
            locals_dict = {
                "inputs": context.get("inputs", {}),
                "results": context.get("results", {}),
                "variables": context.get("variables", {}),
                "context": context.get("context", {})
            }
            
            # 替换变量引用
            # 例如: {{results.step1.output}} -> context["results"]["step1"]["output"]
            import re
            
            def replace_var(match):
                var_path = match.group(1).strip()
                parts = var_path.split(".")
                
                current = locals_dict
                for part in parts:
                    if isinstance(current, dict) and part in current:
                        current = current[part]
                    else:
                        return "None"
                
                return repr(current)
            
            # 替换{{variable}}格式的变量
            condition = re.sub(r"\{\{\s*(.+?)\s*\}\}", replace_var, condition)
            
            # 评估表达式
            result = eval(condition, {"__builtins__": {}}, locals_dict)
            return bool(result)
            
        except Exception as e:
            print(f"条件评估失败: {condition}, 错误: {e}")
            return False
    
    def _prepare_step_params(
        self,
        params: Dict[str, Any],
        context: Dict[str, Any]
    ) -> Dict[str, Any]:
        """准备步骤参数"""
        prepared_params = {}
        
        for key, value in params.items():
            if isinstance(value, str) and value.startswith("{{") and value.endswith("}}"):
                # 变量引用
                var_path = value[2:-2].strip()
                prepared_params[key] = self._resolve_variable(var_path, context)
            elif isinstance(value, dict):
                # 递归处理字典
                prepared_params[key] = self._prepare_step_params(value, context)
            elif isinstance(value, list):
                # 递归处理列表
                prepared_params[key] = [
                    self._prepare_step_params(item, context) if isinstance(item, dict) else item
                    for item in value
                ]
            else:
                prepared_params[key] = value
        
        return prepared_params
    
    def _resolve_variable(self, var_path: str, context: Dict[str, Any]) -> Any:
        """解析变量路径"""
        parts = var_path.split(".")
        
        current = context
        for part in parts:
            if isinstance(current, dict) and part in current:
                current = current[part]
            else:
                # 尝试从嵌套结构中获取
                try:
                    for sub_part in part.split("["):
                        if "]" in sub_part:
                            # 数组索引
                            idx = int(sub_part.split("]")[0])
                            current = current[idx]
                        else:
                            current = current[sub_part]
                except (KeyError, IndexError, TypeError):
                    raise ValueError(f"无法解析变量: {var_path}")
        
        return current
    
    def _get_tool_function(self, tool_name: str) -> Optional[Callable]:
        """获取工具函数"""

        from .decorators import get_tool_registry
        
        registry = get_tool_registry()
        tool = registry.get_tool_by_name(tool_name)
        if not tool:
            return None
        
        # 在实际实现中，这里应该返回工具的实际函数
        # 这里返回一个模拟函数
        async def mock_tool_function(**kwargs):
            return {"status": "success", "data": kwargs}
        
        return mock_tool_function
    
    def _generate_output(
        self,
        chain_def: ChainDefinition,
        context: Dict[str, Any]
    ) -> Any:
        """生成工具链输出"""
        output_schema = chain_def.output_schema
        
        if not output_schema:
            # 如果没有定义输出schema，返回所有步骤结果
            return {
                step_id: result.output
                for step_id, result in context["results"].items()
                if result.status == StepStatus.SUCCESS
            }
        
        # 根据输出schema生成输出
        output = {}
        for field, field_schema in output_schema.get("properties", {}).items():
            # 从field_schema中获取映射规则
            mapping = field_schema.get("mapping")
            if mapping and isinstance(mapping, str):
                # 解析映射规则
                output[field] = self._resolve_variable(mapping, context)
            else:
                # 默认映射到同名字段
                output[field] = context.get("variables", {}).get(field)
        
        return output
    
    async def cancel_chain(self, chain_id: str) -> bool:
        """取消工具链执行"""
        async with self._lock:
            task = self.running_chains.get(chain_id)
            if task:
                task.cancel()
                return True
            return False
    
    def get_chain_status(self, chain_id: str) -> Optional[ChainResult]:
        """获取工具链状态"""
        # 检查是否正在运行
        async with self._lock:
            if chain_id in self.running_chains:
                return ChainResult(
                    chain_id=chain_id,
                    status=ChainStatus.RUNNING,
                    steps={},
                    started_at=datetime.now()
                )
        
        # 检查是否有结果
        return self.chain_results.get(chain_id)
    
    def list_running_chains(self) -> List[str]:
        """列出正在运行的工具链"""
        return list(self.running_chains.keys())
    
    def list_completed_chains(
        self,
        limit: int = 100,
        offset: int = 0
    ) -> List[ChainResult]:
        """列出已完成的工具链"""
        results = list(self.chain_results.values())
        results.sort(key=lambda x: x.completed_at or datetime.min, reverse=True)
        return results[offset:offset + limit]
```

```python
class ConditionOperator(str, Enum):
    """条件运算符"""
    EQUALS = "equals"        # 等于
    NOT_EQUALS = "not_equals" # 不等于
    GREATER_THAN = "greater_than" # 大于

@dataclass
class Condition:
    """条件定义"""
    field: str               # 字段名
    operator: ConditionOperator  # 运算符
    value: Any               # 比较值
    
    def evaluate(self, data: Dict[str, Any]) -> bool:
        """评估条件"""
        field_value = data.get(self.field)
        
        if self.operator == ConditionOperator.EQUALS:
            return field_value == self.value
        elif self.operator == ConditionOperator.NOT_EQUALS:
            return field_value != self.value
        elif self.operator == ConditionOperator.GREATER_THAN:
            return field_value > self.value
        elif self.operator == ConditionOperator.LESS_THAN:
            return field_value < self.value
        elif self.operator == ConditionOperator.CONTAINS:
            return self.value in field_value if field_value else False
        elif self.operator == ConditionOperator.NOT_CONTAINS:
            return self.value not in field_value if field_value else True
        elif self.operator == ConditionOperator.STARTS_WITH:
            return field_value.startswith(self.value) if field_value else False
        elif self.operator == ConditionOperator.ENDS_WITH:
            return field_value.endswith(self.value) if field_value else False
        elif self.operator == ConditionOperator.IS_EMPTY:
            return not field_value or (isinstance(field_value, str) and not field_value.strip())
        elif self.operator == ConditionOperator.IS_NOT_EMPTY:
            return bool(field_value) and (not isinstance(field_value, str) or field_value.strip())
        else:
            return False

@dataclass
class WorkflowDefinition:
    """工作流定义"""
    id: str
    name: str
    description: Optional[str] = None
    version: str = "1.0.0"
    status: WorkflowStatus = WorkflowStatus.DRAFT
    triggers: List[Trigger] = field(default_factory=list)
    steps: List[WorkflowStep] = field(default_factory=list)
    variables: Dict[str, Any] = field(default_factory=dict)  # 工作流变量
    input_schema: Dict[str, Any] = field(default_factory=dict)
    output_schema: Dict[str, Any] = field(default_factory=dict)
    created_at: datetime = field(default_factory=datetime.now)
    updated_at: datetime = field(default_factory=datetime.now)
    
    def validate(self) -> List[str]:
        """验证工作流定义"""
        errors = []
        
        # 检查ID
        if not self.id or not self.id.strip():
            errors.append("工作流ID不能为空")
        
        # 检查名称
        if not self.name or not self.name.strip():
            errors.append("工作流名称不能为空")
        
        # 检查步骤
        if not self.steps:
            errors.append("工作流必须包含至少一个步骤")
        
        # 检查步骤ID唯一性
        step_ids = [step.id for step in self.steps]
        if len(step_ids) != len(set(step_ids)):
            errors.append("步骤ID必须唯一")
        
        # 检查跳转目标是否存在
        all_step_ids = set(step_ids)
        for step in self.steps:
            for target_id in step.on_success + step.on_failure:
                if target_id not in all_step_ids:
                    errors.append(f"步骤 '{step.id}' 的跳转目标 '{target_id}' 不存在")
        
        # 检查是否有循环依赖
        if self._has_cycle():
            errors.append("工作流存在循环依赖")
        
        return errors
    
    def _has_cycle(self) -> bool:
        """检查工作流是否有循环依赖"""
        graph = nx.DiGraph()
        
        # 添加节点
        for step in self.steps:
            graph.add_node(step.id)
        
        # 添加边
        for step in self.steps:
            # 成功跳转
            for target_id in step.on_success:
                graph.add_edge(step.id, target_id)
            
            # 失败跳转
            for target_id in step.on_failure:
                graph.add_edge(step.id, target_id)
        
        # 检查是否有环
        try:
            nx.find_cycle(graph)
            return True
        except nx.NetworkXNoCycle:
            return False
    
    def get_start_steps(self) -> List[WorkflowStep]:
        """获取起始步骤（没有前置依赖的步骤）"""
        # 构建依赖图
        dependencies = {step.id: set() for step in self.steps}
        
        for step in self.steps:
            for target_id in step.on_success + step.on_failure:
                if target_id in dependencies:
                    dependencies[target_id].add(step.id)
        
        # 找到没有依赖的步骤
        start_steps = []
        for step in self.steps:
            if not dependencies[step.id]:
                start_steps.append(step)
        
        return start_steps
    
    def to_dict(self) -> Dict[str, Any]:
        """转换为字典"""
        return {

            "description": self.description,

class WorkflowEngine:
    """工作流引擎"""
    
    def __init__(self, tool_manager, chain_executor):
        self.tool_manager = tool_manager
        self.chain_executor = chain_executor
        self.workflows: Dict[str, WorkflowDefinition] = {}
        self.running_workflows: Dict[str, asyncio.Task] = {}
        self.workflow_results: Dict[str, Dict[str, Any]] = {}
        self._lock = asyncio.Lock()
    
    def register_workflow(self, workflow: WorkflowDefinition) -> None:
        """注册工作流"""
        # 验证工作流
        errors = workflow.validate()
        if errors:
            raise ValueError(f"工作流验证失败: {', '.join(errors)}")
        
        self.workflows[workflow.id] = workflow
    
    def get_workflow(self, workflow_id: str) -> Optional[WorkflowDefinition]:
        """获取工作流"""
        return self.workflows.get(workflow_id)
    
    def list_workflows(
        self,
        status: Optional[WorkflowStatus] = None,
        limit: int = 100,
        offset: int = 0
    ) -> List[WorkflowDefinition]:
        """列出工作流"""
        workflows = list(self.workflows.values())
        
        # 状态过滤
        if status:
            workflows = [w for w in workflows if w.status == status]
        
        # 排序（按更新时间倒序）
        workflows.sort(key=lambda w: w.updated_at, reverse=True)
        
        # 分页
        return workflows[offset:offset + limit]
    
    async def execute_workflow(
        self,
        workflow_id: str,
        inputs: Dict[str, Any],
        context: Optional[Dict[str, Any]] = None
    ) -> Dict[str, Any]:
        """执行工作流"""
        workflow = self.get_workflow(workflow_id)
        if not workflow:
            raise ValueError(f"工作流不存在: {workflow_id}")
        
        if workflow.status != WorkflowStatus.ACTIVE:
            raise ValueError(f"工作流状态为 {workflow.status.value}，无法执行")
        
        # 创建执行ID
        execution_id = f"{workflow_id}_{datetime.now().strftime('%Y%m%d_%H%M%S_%f')}"
        
        # 创建执行任务
        task = asyncio.create_task(
            self._execute_workflow_internal(execution_id, workflow, inputs, context or {})
        )
        
        # 保存运行状态
        async with self._lock:
            self.running_workflows[execution_id] = task
        
        try:
            # 等待任务完成
            result = await task
            return result
        finally:
            # 清理运行状态
            async with self._lock:
                if execution_id in self.running_workflows:
                    del self.running_workflows[execution_id]
    
    async def _execute_workflow_internal(
        self,
        execution_id: str,
        workflow: WorkflowDefinition,
        inputs: Dict[str, Any],
        context: Dict[str, Any]
    ) -> Dict[str, Any]:
        """内部执行工作流"""
        start_time = datetime.now()
        
        # 初始化执行上下文
        execution_context = {
            "execution_id": execution_id,
            "workflow_id": workflow.id,
            "inputs": inputs,
            "context": context,
            "variables": workflow.variables.copy(),
            "step_results": {},
            "current_step": None,
            "status": "running",
            "started_at": start_time.isoformat()
        }
        
        try:
            # 验证输入
            validation_errors = self._validate_workflow_inputs(inputs, workflow.input_schema)
            if validation_errors:
                execution_context["status"] = "failed"
                execution_context["error"] = f"输入验证失败: {', '.join(validation_errors)}"
                execution_context["completed_at"] = datetime.now().isoformat()
                self.workflow_results[execution_id] = execution_context
                return execution_context
            
            # 获取起始步骤
            start_steps = workflow.get_start_steps()
            if not start_steps:
                execution_context["status"] = "failed"
                execution_context["error"] = "工作流没有起始步骤"
                execution_context["completed_at"] = datetime.now().isoformat()
                self.workflow_results[execution_id] = execution_context
                return execution_context
            
            # 执行工作流
            execution_context = await self._execute_workflow_steps(
                workflow, start_steps, execution_context
            )
            
            # 生成输出
            if execution_context["status"] == "completed":
                execution_context["output"] = self._generate_workflow_output(
                    workflow, execution_context
                )
            
        except Exception as e:
            execution_context["status"] = "failed"
            execution_context["error"] = f"工作流执行异常: {str(e)}"
        finally:
            # 计算执行时间
            end_time = datetime.now()
            execution_context["completed_at"] = end_time.isoformat()
            execution_context["execution_time"] = (end_time - start_time).total_seconds()
            
            # 保存结果
            self.workflow_results[execution_id] = execution_context
        
        return execution_context
    
    def _validate_workflow_inputs(
        self,
        inputs: Dict[str, Any],
        input_schema: Dict[str, Any]
    ) -> List[str]:
        """验证工作流输入"""
        # 简化实现，实际应该使用JSON Schema验证
        errors = []
        
        required_fields = input_schema.get("required", [])
        for field in required_fields:
            if field not in inputs:
                errors.append(f"缺少必需字段: {field}")
        
        return errors
    
    async def _execute_workflow_steps(
        self,
        workflow: WorkflowDefinition,
        current_steps: List[WorkflowStep],
        context: Dict[str, Any]
    ) -> Dict[str, Any]:
        """执行工作流步骤"""
        # 创建步骤执行任务
        step_tasks = []
        for step in current_steps:
            task = asyncio.create_task(
                self._execute_workflow_step(workflow, step, context)
            )
            step_tasks.append((step.id, task))
        
        # 等待所有步骤完成
        step_results = {}
        for step_id, task in step_tasks:
            try:
                result = await task
                step_results[step_id] = result
            except Exception as e:
                step_results[step_id] = {
                    "status": "failed",
                    "error": str(e),
                    "step_id": step_id
                }
        
        # 更新上下文
        context["step_results"].update(step_results)
        
        # 检查执行结果
        all_completed = True
        next_steps = []
        
        for step_id, result in step_results.items():
            if result["status"] == "success":
                # 成功时跳转
                step = next((s for s in workflow.steps if s.id == step_id), None)
                if step:
                    next_steps.extend([
                        next((s for s in workflow.steps if s.id == target_id), None)
                        for target_id in step.on_success
                    ])
            elif result["status"] == "failed":
                # 失败时跳转
                step = next((s for s in workflow.steps if s.id == step_id), None)
                if step:
                    next_steps.extend([
                        next((s for s in workflow.steps if s.id == target_id), None)
                        for target_id in step.on_failure
                    ])
            else:
                all_completed = False
        
        # 过滤掉None（不存在的步骤）
        next_steps = [step for step in next_steps if step is not None]
        
        # 如果没有下一步，工作流完成
        if not next_steps:
            context["status"] = "completed"
            return context
        
        # 如果有下一步，继续执行
        if all_completed:
            # 递归执行下一步
            return await self._execute_workflow_steps(workflow, next_steps, context)
        else:
            # 有步骤还在执行，等待
            context["status"] = "running"
            return context
    
    async def _execute_workflow_step(
        self,
        workflow: WorkflowDefinition,
        step: WorkflowStep,
        context: Dict[str, Any]
    ) -> Dict[str, Any]:
        """执行工作流步骤"""
        step_start_time = datetime.now()
        
        result = {
            "step_id": step.id,
            "step_name": step.name,
            "status": "pending",
            "started_at": step_start_time.isoformat(),
            "tool": step.tool
        }
        
        try:
            # 检查执行条件
            if step.conditions:
                conditions_met = all(
                    condition.evaluate(context["variables"])
                    for condition in step.conditions
                )
                if not conditions_met:
                    result["status"] = "skipped"
                    result["completed_at"] = datetime.now().isoformat()
                    result["execution_time"] = 0.0
                    return result
            
            # 准备参数
            params = self._prepare_workflow_step_params(step.params, context)
            
            # 更新上下文
            context["current_step"] = step.id
            
            # 执行工具或工具链
            if step.tool.startswith("chain:"):
                # 执行工具链
                chain_id = step.tool[6:]  # 去掉"chain:"前缀
                chain_result = await self.chain_executor.execute_chain(
                    chain_id=chain_id,
                    inputs=params,
                    context=context
                )
                
                result["output"] = chain_result.output
                result["status"] = chain_result.status.value
                if chain_result.error:
                    result["error"] = chain_result.error
                
            else:
                # 执行单个工具
                # 这里应该调用工具执行器
                # 简化实现，直接返回模拟结果
                result["output"] = {"params": params, "context": context}
                result["status"] = "success"
            
            # 更新工作流变量
            if result["status"] == "success" and result.get("output"):
                context["variables"][f"step_{step.id}_output"] = result["output"]
            
        except Exception as e:
            result["status"] = "failed"
            result["error"] = str(e)
        finally:
            # 计算执行时间
            step_end_time = datetime.now()
            result["completed_at"] = step_end_time.isoformat()
            result["execution_time"] = (step_end_time - step_start_time).total_seconds()
        
        return result
    
    def _prepare_workflow_step_params(
        self,
        params: Dict[str, Any],
        context: Dict[str, Any]
    ) -> Dict[str, Any]:
        """准备工作流步骤参数"""
        # 与工具链类似的参数准备逻辑
        # 支持变量替换：{{variable.path}}
        
        def resolve_value(value):
            if isinstance(value, str) and value.startswith("{{") and value.endswith("}}"):
                # 变量引用
                var_path = value[2:-2].strip()
                return self._resolve_workflow_variable(var_path, context)
            elif isinstance(value, dict):
                # 递归处理字典
                return {k: resolve_value(v) for k, v in value.items()}
            elif isinstance(value, list):
                # 递归处理列表
                return [resolve_value(item) for item in value]
            else:
                return value
        
        return {k: resolve_value(v) for k, v in params.items()}
    
    def _resolve_workflow_variable(self, var_path: str, context: Dict[str, Any]) -> Any:
        """解析工作流变量"""
        # 支持多种变量来源：
        # inputs.field - 工作流输入
        # variables.field - 工作流变量
        # step_{step_id}_output.field - 步骤输出
        # context.field - 执行上下文
        
        parts = var_path.split(".")
        if len(parts) < 2:
            raise ValueError(f"无效的变量路径: {var_path}")
        
        source = parts[0]
        path = parts[1:]
        
        if source == "inputs":
            current = context.get("inputs", {})
        elif source == "variables":
            current = context.get("variables", {})
        elif source.startswith("step_") and source.endswith("_output"):
            step_id = source[5:-7]  # 去掉"step_"和"_output"
            current = context.get("step_results", {}).get(step_id, {}).get("output", {})
        elif source == "context":
            current = context.get("context", {})
        else:
            raise ValueError(f"未知的变量来源: {source}")
        
        # 遍历路径
        for part in path:
            if isinstance(current, dict) and part in current:
                current = current[part]
            elif isinstance(current, list):
                try:
                    idx = int(part)
                    current = current[idx]
                except (ValueError, IndexError):
                    raise ValueError(f"无法解析变量路径: {var_path}")
            else:
                raise ValueError(f"无法解析变量路径: {var_path}")
        
        return current
    
    def _generate_workflow_output(
        self,
        workflow: WorkflowDefinition,
        context: Dict[str, Any]
    ) -> Any:
        """生成工作流输出"""
        output_schema = workflow.output_schema
        
        if not output_schema:
            # 如果没有定义输出schema，返回所有步骤结果
            return {
                "step_results": context["step_results"],
                "variables": context["variables"]
            }
        
        # 根据输出schema生成输出
        output = {}
        for field, field_schema in output_schema.get("properties", {}).items():
            mapping = field_schema.get("mapping")
            if mapping:
                output[field] = self._resolve_workflow_variable(mapping, context)
            else:
                output[field] = context.get("variables", {}).get(field)
        
        return output
    
    async def cancel_workflow(self, execution_id: str) -> bool:
        """取消工作流执行"""
        async with self._lock:
            task = self.running_workflows.get(execution_id)
            if task:
                task.cancel()
                return True
            return False
    
    def get_workflow_status(self, execution_id: str) -> Optional[Dict[str, Any]]:
        """获取工作流执行状态"""
        # 检查是否正在运行
        async with self._lock:
            if execution_id in self.running_workflows:
                return {
                    "execution_id": execution_id,
                    "status": "running",
                    "started_at": datetime.now().isoformat()
                }
        
        # 检查是否有结果
        return self.workflow_results.get(execution_id)
    
    def list_running_workflows(self) -> List[str]:
        """列出正在运行的工作流"""
        return list(self.running_workflows.keys())
    
    def list_completed_workflows(
        self,
        workflow_id: Optional[str] = None,
        limit: int = 100,
        offset: int = 0
    ) -> List[Dict[str, Any]]:
        """列出已完成的工作流"""
        results = list(self.workflow_results.values())
        
        # 工作流ID过滤
        if workflow_id:
            results = [r for r in results if r.get("workflow_id") == workflow_id]
        
        # 排序（按完成时间倒序）
        results.sort(
            key=lambda x: x.get("completed_at") or "0000-00-00T00:00:00",
            reverse=True
        )
        
        # 分页
        return results[offset:offset + limit]
    
    def export_workflow(self, workflow_id: str, format: str = "json") -> str:
        """导出工作流定义"""
        workflow = self.get_workflow(workflow_id)
        if not workflow:
            raise ValueError(f"工作流不存在: {workflow_id}")
        
        data = workflow.to_dict()
        
        if format == "json":
            return json.dumps(data, ensure_ascii=False, indent=2)
        elif format == "yaml":
            return yaml.dump(data, allow_unicode=True, sort_keys=False)
        else:
            raise ValueError(f"不支持的格式: {format}")
    
    def import_workflow(self, data: Union[str, Dict]) -> str:
        """导入工作流定义"""
        if isinstance(data, str):
            try:
                data = json.loads(data)
            except json.JSONDecodeError:
                data = yaml.safe_load(data)
        
        workflow = WorkflowDefinition.from_dict(data)
        
        # 验证工作流
        errors = workflow.validate()
        if errors:
            raise ValueError(f"工作流验证失败: {', '.join(errors)}")
        
        # 注册工作流
        self.register_workflow(workflow)
        
        return workflow.id
```

```python
        async def wrapper(*args, **kwargs):

            if not user:
                # 尝试从请求上下文获取
                from app.utils.context import get_current_user
                user = get_current_user()
            
            if not user:
                raise PermissionError("未认证的用户")
            
            # 获取资源ID
            if resource_id_getter:
                resource_id = resource_id_getter(*args, **kwargs)
            else:
                # 默认从kwargs中获取
                resource_id = kwargs.get("resource_id") or kwargs.get("tool_id") or kwargs.get("chain_id") or kwargs.get("workflow_id")
            
            if not resource_id:
                raise ValueError("无法确定资源ID")
            
            # 检查权限
            permission_manager = PermissionManager()  # 实际应该使用单例
            if not permission_manager.check_permission(user, resource_type, resource_id, required_permission):
                raise PermissionError(f"用户没有{required_permission.value}权限访问资源{resource_type.value}:{resource_id}")
            
            # 执行函数
            return await func(*args, **kwargs)
        
        return wrapper
    
    return decorator
```

        try:
            if os.path.exists(work_dir):
                shutil.rmtree(work_dir)
            if work_dir in self._temp_dirs:
                self._temp_dirs.remove(work_dir)
        except Exception as e:
            print(f"清理工作目录失败: {e}")
    
    def _is_command_allowed(self, command: str) -> bool:
        """检查命令是否允许执行"""
        # 检查阻止的命令
        for blocked in self.config.blocked_commands:
            if blocked in command:
                return False
        
        # 检查允许的命令
        if self.config.allowed_commands:
            for allowed in self.config.allowed_commands:
                if allowed in command:
                    return True
            return False
        
        return True
    
    def _get_resource_limits(self) -> Dict[str, Any]:
        """获取资源限制配置"""
        limits = {}
        
        # 内存限制
        if self.config.memory_limit_mb > 0:
            limits["memory"] = self.config.memory_limit_mb * 1024 * 1024  # 转换为字节
        
        # CPU限制
        if self.config.cpu_limit_percent > 0:
            limits["cpu_percent"] = self.config.cpu_limit_percent
        
        # 磁盘限制
        if self.config.disk_limit_mb > 0:
            limits["disk"] = self.config.disk_limit_mb * 1024 * 1024  # 转换为字节
        
        return limits
    
    async def _run_in_sandbox(
        self,
        command: List[str],
        work_dir: str,
        input_data: Optional[str],
        env: Optional[Dict[str, str]],
        resource_limits: Dict[str, Any]
    ) -> SandboxResult:
        """在沙箱中运行命令"""
        # 准备环境变量
        process_env = os.environ.copy()
        if env:
            process_env.update(env)
        
        # 设置工作RR目录
        if self.config.working_directory:
            work_dir = self.config.working_directory
        
        # 执行命令
        try:
            # 这里应该使用实际的沙箱执行实现
            # 例如使用subprocess或容器技术
            import subprocess
            import time
            
            start_time = time.time()
            process = subprocess.run(
                command,
                cwd=work_dir,
                env=process_env,
                input=input_data.encode() if input_data else None,
                capture_output=True,
                text=True,
                timeout=self.config.timeout_seconds
            )
            
            execution_time = time.time() - start_time
            
            return SandboxResult(
                success=process.returncode == 0,
                output=process.stdout,
                error=process.stderr,
                exit_code=process.returncode,
                execution_time=execution_time,
                memory_used_mb=0.0,  # 实际实现中需要监控资源使用
                cpu_used_percent=0.0,
                disk_used_mb=0.0
            )
            
        except subprocess.TimeoutExpired:
            return SandboxResult(
                success=False,
                output="",
                error=f"命令执行超时（{self.config.timeout_seconds}秒）",
                execution_time=self.config.timeout_seconds
            )
        except Exception as e:
            return SandboxResult(
                success=False,
                output="",
                error=f"命令执行失败: {str(e)}",
                execution_time=time.time() - start_time
            )

    def cleanup(self) -> None:
        """清理所有临时资源"""
        for temp_dir in self._temp_dirs[:]:
            self._cleanup_work_directory(temp_dir)


class ToolSecurityManager:
    """工具安全管理器"""
    
    def __init__(self, permission_manager: PermissionManager, sandbox_executor: SandboxExecutor):
        self.permission_manager = permission_manager
        self.sandbox_executor = sandbox_executor
        self._audit_log: List[Dict[str, Any]] = []
    
    async def execute_tool_with_security(
        self,
        user_id: str,
        tool_id: str,
        tool_metadata: ToolMetadata,
        parameters: Dict[str, Any]
    ) -> Dict[str, Any]:
        """在安全控制下执行工具"""
        # 1. 权限检查
        if not self.permission_manager.check_permission(user_id, tool_id, "execute"):
            raise PermissionError(f"用户 {user_id} 没有执行工具 {tool_id} 的权限")
        
        # 2. 参数验证
        self._validate_parameters(tool_metadata, parameters)
        
        # 3. 安全检查
        if not self._check_security_policy(tool_metadata, parameters):
            raise SecurityError(f"工具 {tool_id} 的安全检查失败")
        
        # 4. 记录审计日志
        self._log_audit(user_id, tool_id, parameters, "execute_start")
        
        try:
            # 5. 根据工具类型选择执行方式
            result = await self._execute_tool(tool_metadata, parameters)
            
            # 6. 记录成功审计日志
            self._log_audit(user_id, tool_id, parameters, "execute_success", result)
            
            return result
            
        except Exception as e:
            # 7. 记录失败审计日志
            self._log_audit(user_id, tool_id, parameters, "execute_failed", error=str(e))
            raise
    
    def _validate_parameters(self, tool_metadata: ToolMetadata, parameters: Dict[str, Any]) -> None:
        """验证工具参数"""
        # 检查必需参数
        for param_name, param_spec in tool_metadata.requirements.parameters.items():
            if param_spec.required and param_name not in parameters:
                raise ValueError(f"缺少必需参数: {param_name}")
        
        # 检查参数类型
        for param_name, param_value in parameters.items():
            if param_name in tool_metadata.requirements.parameters:
                param_spec = tool_metadata.requirements.parameters[param_name]
                if not self._check_parameter_type(param_value, param_spec.type):
                    raise TypeError(f"参数 {param_name} 类型不匹配: 期望 {param_spec.type}, 实际 {type(param_value)}")
    
    def _check_parameter_type(self, value: Any, expected_type: str) -> bool:
        """检查参数类型"""
        type_map = {
            "string": str,
            "integer": int,
            "number": (int, float),
            "boolean": bool,
            "array": list,
            "object": dict
        }
        
        if expected_type not in type_map:
            return True  # 未知类型，跳过检查
        
        expected = type_map[expected_type]
        if isinstance(expected, tuple):
            return isinstance(value, expected)
        return isinstance(value, expected)
    
    def _check_security_policy(self, tool_metadata: ToolMetadata, parameters: Dict[str, Any]) -> bool:
        """检查安全策略"""
        # 检查工具权限要求
        for permission in tool_metadata.permissions:
            if not self.permission_manager.has_permission(permission):
                return False
        
        # 检查参数安全
        for param_name, param_value in parameters.items():
            if isinstance(param_value, str):
                # 检查SQL注入
                if self._contains_sql_injection(param_value):
                    return False
                
                # 检查命令注入
                if self._contains_command_injection(param_value):
                    return False
        
        return True
    
    def _contains_sql_injection(self, value: str) -> bool:
        """检查是否包含SQL注入"""
        sql_keywords = ["SELECT", "INSERT", "UPDATE", "DELETE", "DROP", "UNION", "OR", "AND"]
        value_upper = value.upper()
        for keyword in sql_keywords:
            if keyword in value_upper:
                return True
        return False
    
    def _contains_command_injection(self, value: str) -> bool:
        """检查是否包含命令注入"""
        dangerous_chars = [";", "|", "&", "`", "$", "(", ")", "<", ">"]
        for char in dangerous_chars:
            if char in value:
                return True
        return False
    
    async def _execute_tool(self, tool_metadata: ToolMetadata, parameters: Dict[str, Any]) -> Dict[str, Any]:
        """执行工具"""
        # 根据工具类型选择执行方式
        if tool_metadata.type == ToolType.SYSTEM_COMMAND:
            # 系统命令在沙箱中执行
            command = parameters.get("command", "")
            args = parameters.get("args", [])
            return await self._execute_system_command(command, args)
        
        elif tool_metadata.type == ToolType.FILE_OPERATION:
            # 文件操作需要特殊权限检查
            return await self._execute_file_operation(parameters)
        
        else:
            # 其他工具类型直接调用
            return await self._call_tool_function(tool_metadata, parameters)
    
    async def _execute_system_command(self, command: str, args: List[str]) -> Dict[str, Any]:
        """在沙箱中执行系统命令"""
        result = await self.sandbox_executor.execute_command(command, args)
        
        return {
            "success": result.success,
            "output": result.output,
            "error": result.error,
            "exit_code": result.exit_code,
            "execution_time": result.execution_time
        }
    
    async def _execute_file_operation(self, parameters: Dict[str, Any]) -> Dict[str, Any]:
        """执行文件操作"""
        operation = parameters.get("operation", "")
        path = parameters.get("path", "")
        
        # 检查路径安全性
        if not self._is_safe_path(path):
            raise SecurityError(f"不安全的文件路径: {path}")
        
        # 执行文件操作
        if operation == "read":
            with open(path, "r", encoding="utf-8") as f:
                content = f.read()
            return {"success": True, "content": content}
        
        elif operation == "write":
            content = parameters.get("content", "")
            with open(path, "w", encoding="utf-8") as f:
                f.write(content)
            return {"success": True}
        
        else:
            raise ValueError(f"不支持的文件操作: {operation}")
    
    def _is_safe_path(self, path: str) -> bool:
        """检查路径是否安全"""
        # 防止路径遍历攻击
        if ".." in path or path.startswith("/") or ":" in path:
            return False
        
        # 限制访问特定目录
        allowed_dirs = ["/tmp", "/var/tmp", "/home/user/data"]
        for allowed_dir in allowed_dirs:
            if path.startswith(allowed_dir):
                return True
        
        return False
    
    async def _call_tool_function(self, tool_metadata: ToolMetadata, parameters: Dict[str, Any]) -> Dict[str, Any]:
        """调用工具函数"""
        # 这里应该调用实际的工具函数
        # 简化实现，返回模拟结果
        return {
            "success": True,
            "result": f"执行工具 {tool_metadata.name} 成功",
            "execution_time": 0.1
        }
    
    def _log_audit(self, user_id: str, tool_id: str, parameters: Dict[str, Any], 
                   action: str, result: Optional[Dict[str, Any]] = None, error: Optional[str] = None) -> None:
        """记录审计日志"""
        log_entry = {
            "timestamp": datetime.now().isoformat(),
            "user_id": user_id,
            "tool_id": tool_id,
            "action": action,
            "parameters": self._sanitize_parameters(parameters),
            "result": result,
            "error": error
        }
        
        self._audit_log.append(log_entry)
        
        # 限制日志大小
        if len(self._audit_log) > 1000:
            self._audit_log = self._audit_log[-1000:]
    
    def _sanitize_parameters(self, parameters: Dict[str, Any]) -> Dict[str, Any]:
        """清理敏感参数"""
        sanitized = {}
        sensitive_keys = ["password", "token", "secret", "key", "credential"]
        
        for key, value in parameters.items():
            if any(sensitive in key.lower() for sensitive in sensitive_keys):
                sanitized[key] = "***REDACTED***"
            else:
                sanitized[key] = value
        
        return sanitized
    
    def get_audit_logs(self, limit: int = 100) -> List[Dict[str, Any]]:
        """获取审计日志"""
        return self._audit_log[-limit:]


class SecurityMiddleware:
    """安全中间件"""
    
    def __init__(self, security_manager: ToolSecurityManager):
        self.security_manager = security_manager
    
    async def __call__(self, request: Dict[str, Any], call_next) -> Dict[str, Any]:
        """中间件处理"""
        # 1. 验证请求
        if not self._validate_request(request):
            return {"error": "无效请求", "success": False}
        
        # 2. 提取用户信息
        user_id = request.get("user_id", "anonymous")
        
        # 3. 检查工具权限
        tool_id = request.get("tool_id")
        if not tool_id:
            return {"error": "缺少工具ID", "success": False}
        
        # 4. 执行工具
        try:
            result = await self.security_manager.execute_tool_with_security(
                user_id=user_id,
                tool_id=tool_id,
                tool_metadata=request.get("tool_metadata"),
                parameters=request.get("parameters", {})
            )
            
            # 5. 添加安全头
            result["security_headers"] = {
                "x-content-type-options": "nosniff",
                "x-frame-options": "DENY",
                "x-xss-protection": "1; mode=block"
            }
            
            return result
            
        except PermissionError as e:
            return {"error": str(e), "success": False, "code": "PERMISSION_DENIED"}
        except SecurityError as e:
            return {"error": str(e), "success": False, "code": "SECURITY_VIOLATION"}
        except Exception as e:
            return {"error": f"执行失败: {str(e)}", "success": False, "code": "EXECUTION_ERROR"}
    
    def _validate_request(self, request: Dict[str, Any]) -> bool:
        """验证请求"""
        required_fields = ["tool_id", "parameters"]
        for field in required_fields:
            if field not in request:
                return False
        
        # 检查参数类型
        if not isinstance(request["parameters"], dict):
            return False
        
        return True

```python
class ConfigManager:
    """配置管理器"""
    
    def __init__(self, config_path: str):
        self.config_path = config_path
        self.config = self._load_config()
        self._last_modified = os.path.getmtime(config_path)
        
    def _load_config(self) -> Dict[str, Any]:
        """加载配置"""
        with open(self.config_path, 'r', encoding='utf-8') as f:
            return yaml.safe_load(f)
    
    def check_and_reload(self) -> bool:
        """检查并重新加载配置"""
        current_modified = os.path.getmtime(self.config_path)
        if current_modified > self._last_modified:
            self.config = self._load_config()
            self._last_modified = current_modified
            return True
        return False
    
    def get(self, key: str, default: Any = None) -> Any:
        """获取配置值"""
        keys = key.split('.')
        value = self.config
        
        for k in keys:
            if isinstance(value, dict) and k in value:
                value = value[k]
            else:
                return default
        
        return value
```

### 8.7.1 缓存策略
```python
from functools import lru_cache
import time

class ToolCache:
    """工具缓存"""
    
    def __init__(self, max_size: int = 1000, ttl: int = 3600):
        self.cache = {}
        self.max_size = max_size
        self.ttl = ttl  # 秒
    
    @lru_cache(maxsize=100)
    def get_tool(self, tool_id: str) -> Optional[ToolMetadata]:
        """获取工具（带缓存）"""
        if tool_id in self.cache:
            cached_item = self.cache[tool_id]
            if time.time() - cached_item["timestamp"] < self.ttl:
                return cached_item["tool"]
        
        # 从数据库或文件加载
        tool = self._load_tool(tool_id)
        if tool:
            self.cache[tool_id] = {
                "tool": tool,
                "timestamp": time.time()
            }
            # 清理过期缓存
            self._cleanup_cache()
        
        return tool
    
    def _cleanup_cache(self):
        """清理缓存"""
        if len(self.cache) > self.max_size:
            # 移除最旧的缓存项
            sorted_items = sorted(self.cache.items(), key=lambda x: x[1]["timestamp"])
            for tool_id, _ in sorted_items[:len(self.cache) - self.max_size]:
                del self.cache[tool_id]
```

```python
import asyncio
from concurrent.futures import ThreadPoolExecutor, ProcessPoolExecutor

class AsyncExecutor:
    """异步执行器"""
    
    def __init__(self, max_workers: int = 10):
        self.thread_pool = ThreadPoolExecutor(max_workers=max_workers)
        self.process_pool = ProcessPoolExecutor(max_workers=max_workers)
    
    async def execute_io_bound(self, func, *args, **kwargs):
        """执行I/O密集型任务"""
        loop = asyncio.get_event_loop()
        return await loop.run_in_executor(
            self.thread_pool,

    async def execute_cpu_bound(self, func, *args, **kwargs):
        """执行CPU密集型任务"""
        loop = asyncio.get_event_loop()
        return await loop.run_in_executor(
            self.process_pool,
```

```python
import json
from datetime import datetime

class ToolRegistryBackup:
    """工具注册表备份"""
    
    def __init__(self, registry: ToolRegistry, backup_dir: str = "backups"):
        self.registry = registry
        self.backup_dir = backup_dir
        os.makedirs(backup_dir, exist_ok=True)
    
    def create_backup(self) -> str:
        """创建备份"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        backup_file = os.path.join(self.backup_dir, f"tool_registry_{timestamp}.json")
        
        # 导出所有工具
        tools_data = []
        for tool_id, tool in self.registry._tools.items():
            tools_data.append({
                "id": tool_id,
                "name": tool.name,
                "display_name": tool.display_name,
                "description": tool.description,
                "type": tool.type.value,
                "category": tool.category,
                "capabilities": [c.value for c in tool.capabilities],
                "permissions": [p.to_dict() for p in tool.permissions],
                "requirements": tool.requirements.to_dict() if tool.requirements else None
            })
        
        # 保存到文件
        with open(backup_file, 'w', encoding='utf-8') as f:
            json.dump({
                "timestamp": timestamp,
                "tool_count": len(tools_data),
                "tools": tools_data
            }, f, indent=2, ensure_ascii=False)
        
        return backup_file
    
    def restore_backup(self, backup_file: str) -> bool:
        """恢复备份"""
        try:
            with open(backup_file, 'r', encoding='utf-8') as f:
                backup_data = json.load(f)
            
            # 清空当前注册表
            self.registry._tools.clear()
            self.registry._name_index.clear()
            self.registry._display_name_index.clear()
            self.registry._categories.clear()
            self.registry._types.clear()
            self.registry._capabilities.clear()
            
            # 恢复工具
            for tool_data in backup_data["tools"]:
                tool = ToolMetadata(
                    name=tool_data["name"],
                    display_name=tool_data["display_name"],
                    description=tool_data["description"],
                    type=ToolType(tool_data["type"]),
                    category=tool_data.get("category"),
                    capabilities=[ToolCapability(c) for c in tool_data.get("capabilities", [])],
                    permissions=[ToolPermission.from_dict(p) for p in tool_data.get("permissions", [])],
                    requirements=ToolRequirements.from_dict(tool_data["requirements"]) if tool_data.get("requirements") else None
                )
                
                self.registry.register(tool_data["id"], tool)
            
            return True
            
        except Exception as e:
            print(f"恢复备份失败: {e}")
            return False
```

```python
class MonitoringSystem:
    """监控系统"""
    
    def __init__(self):
        self.metrics = {}
        self.alerts = []
    
    def record_metric(self, name: str, value: float, tags: Dict[str, str] = None):
        """记录指标"""
        if name not in self.metrics:
            self.metrics[name] = []
        
        self.metrics[name].append({
            "timestamp": datetime.now().isoformat(),
            "value": value,
            "tags": tags or {}
        })
        
        # 检查告警条件
        self._check_alerts(name, value, tags)
    
    def add_alert(self, name: str, condition, message: str, severity: str = "warning"):
        """添加告警规则"""
        self.alerts.append({
            "name": name,
            "condition": condition,
            "message": message,
            "severity": severity
        })
    
    def _check_alerts(self, metric_name: str, value: float, tags: Dict[str, str]):
        """检查告警条件"""
        for alert in self.alerts:
            if alert["condition"](metric_name, value, tags):
                self._trigger_alert(alert, value, tags)
    
    def _trigger_alert(self, alert: Dict[str, Any], value: float, tags: Dict[str, str]):
        """触发告警"""
        alert_message = f"[{alert['severity'].upper()}] {alert['message']}"
        alert_message += f" - 当前值: {value}"
        if tags:
            alert_message += f" - 标签: {tags}"
        
        print(f"告警: {alert_message}")
```
