## 2 查询方法详述 【Windows系统网络查询命令】2026-02-15 17:52:30

### 2.1 查询WiFi连接信息

#### 2.1.1 命令
```cmd
netsh wlan show interfaces
```

#### 2.1.2 参数说明
- `netsh`: Windows网络配置命令行工具
- `wlan`: 无线局域网子命令
- `show interfaces`: 显示所有无线接口信息

#### 2.1.3 返回信息说明
| 字段 | 含义 |
|------|------|
| Name | 接口名称（WLAN） |
| Description | 网卡型号 |
| Physical address | MAC地址 |
| SSID | WiFi网络名称 |
| BSSID | 路由器MAC地址 |
| State | 连接状态 |
| Signal | 信号强度（百分比） |
| Band | 频段（2.4/5 GHz） |
| Authentication | 认证方式 |

### 2.3 使用Python查询（跨平台通用方法）

#### 2.3.1 纯Python标准库方法
```python
import socket

# 获取主机名
hostname = socket.gethostname()
print(f'主机名: {hostname}')

# 获取本机IP（方法1：通过主机名解析）
ip_address = socket.gethostbyname(hostname)
print(f'本机IP: {ip_address}')

# 获取本机IP（方法2：通过UDP连接外部获取真实出口IP）
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.connect(('8.8.8.8', 80))  # Google DNS
local_ip = s.getsockname()[0]
s.close()
print(f'局域网IP: {local_ip}')
```

#### 2.3.2 使用 psutil 库（需要安装）
```python
import psutil

# 获取所有网络接口
for interface_name, interface_addresses in psutil.net_if_addrs().items():
    print(f'接口: {interface_name}')
    for address in interface_addresses:
        if address.family == socket.AF_INET:  # IPv4
            print(f'  IPv4: {address.address}')
        elif address.family == socket.AF_INET6:  # IPv6
            print(f'  IPv6: {address.address}')
        elif address.family == psutil.AF_LINK:  # MAC
            print(f'  MAC: {address.address}')
```

### 4.2 Python脚本查询

```python
# 保存为 network_check.py
import socket
import json
import urllib.request

def get_local_ip():
    s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
    s.connect(('8.8.8.8', 80))
    ip = s.getsockname()[0]
    s.close()
    return ip

def get_public_ip():
    try:
        with urllib.request.urlopen('https://api.ipify.org?format=json') as response:
            data = json.loads(response.read().decode())
            return data['ip']
    except:
        return 'Unknown'

print(f'本机局域网IP: {get_local_ip()}')
print(f'公网出口IP: {get_public_ip()}')
```

## 5 注意事项 【查询过程中的经验】2026-02-15 17:52:30

### 5.1 命令行工具选择
- **Windows CMD**: 使用 `ipconfig` 和 `netsh`
- **PowerShell**: 使用 `Get-NetIPAddress` 等Cmdlet
- **Git Bash**: 可以直接使用 `curl` 命令
- **Python**: 跨平台通用，但需要安装依赖库

### 5.2 公网查询服务选择
- **国内环境推荐**: cip.cc、ip-api.com (速度快)
- **国际环境推荐**: ipinfo.io、ipify.org
- **多个服务交叉验证**: 确保信息准确性

### 5.3 可能遇到的问题
- **403错误**: 部分服务会限制访问频率，可更换其他服务
- **连接超时**: 检查网络连接，或更换查询服务
- **信息不一致**: 使用至少2-3个不同服务交叉验证

### 5.4 IP地址类型说明
- **局域网IP (192.168.x.x)**: 仅在本地网络内有效，用于内网通信
- **公网IP (171.93.x.x)**: 对外访问互联网时的身份标识
- **127.0.0.1**: 本地回环地址，用于本机内部通信
