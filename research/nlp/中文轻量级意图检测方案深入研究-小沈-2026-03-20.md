## 二、方案一：Jieba（分词+关键词）

### 2.1 基本信息

| 属性 | 说明 |
|------|------|
| **名称** | Jieba（结巴） |
| **安装** | `pip install jieba` |
| **大小** | ~15MB |
| **延迟** | 10-50ms |
| **Stars** | 33K+ (GitHub) |
| **特点** | 最流行的中文分词工具 |

### 2.2 核心功能

| 功能 | 说明 |
|------|------|
| **分词** | 4种模式：精确模式、全模式、搜索引擎模式、paddle模式 |
| **关键词提取** | TF-IDF + TextRank 两种算法 |
| **词性标注** | 兼容 ictclas 标记法 |
| **自定义词典** | 支持加载用户词典，动态添加/删除词语 |

### 2.3 意图检测实现

```python
import jieba
import jieba.analyse

class JiebaIntentDetector:
    """基于 Jieba 的意图检测器"""
    
    def __init__(self):
        # 意图关键词映射
        self.intent_keywords = {
            "file": ["文件", "读取", "写入", "删除", "移动", "目录", "搜索文件", 
                     "复制", "粘贴", "创建", "重命名", "备份"],
            "network": ["搜索", "下载", "http", "网址", "网络", "请求", 
                        "API", "访问", "打开网页"],
            "system": ["进程", "CPU", "内存", "剪贴板", "系统", "关机", 
                       "重启", "音量", "亮度"],
            "code": ["代码", "执行", "python", "运行", "脚本", "计算"],
        }
        
        # 加载自定义词典（可选）
        # jieba.load_userdict("user_dict.txt")
    
    def detect(self, text: str) -> dict:
        """意图检测"""
        # Step 1: 分词
        words = list(jieba.cut(text))
        
        # Step 2: 关键词提取（TF-IDF）
        keywords = jieba.analyse.extract_tags(text, topK=5)
        
        # Step 3: 计算各意图得分
        scores = {}
        for intent, intent_kws in self.intent_keywords.items():
            score = 0
            # 精确匹配
            for kw in intent_kws:
                if kw in text:
                    score += 2  # 精确匹配权重高
            # 分词匹配
            for word in words:
                if word in intent_kws:
                    score += 1
            # 关键词匹配
            for kw in keywords:
                if kw in intent_kws:
                    score += 1.5
            scores[intent] = score
        
        # Step 4: 选择最佳意图
        if not scores or max(scores.values()) == 0:
            return {"intent": "unknown", "confidence": 0, "words": words, "keywords": keywords}
        
        best_intent = max(scores, key=scores.get)
        confidence = scores[best_intent] / (sum(scores.values()) + 0.01)
        
        return {
            "intent": best_intent,
            "confidence": round(confidence, 2),
            "words": words,
            "keywords": keywords,
        }

# 使用示例
detector = JiebaIntentDetector()
result = detector.detect("帮我读取 D:/test.txt 文件")
print(result)
# {'intent': 'file', 'confidence': 0.85, 'words': ['帮', '我', '读取', ' ', 'D', ':', '/', 'test', '.', 'txt', ' ', '文件'], 'keywords': ['test.txt', '读取', '文件']}
```

### 2.4 优缺点

| 优点 | 缺点 |
|------|------|
| ✅ 最轻量（~15MB） | ❌ 只做分词，没有内置意图分类 |
| ✅ 最快（10-50ms） | ❌ 需要自己写关键词映射逻辑 |
| ✅ 支持自定义词典 | ❌ 语义理解能力弱 |
| ✅ 关键词提取（TF-IDF/TextRank） | ❌ 处理复杂表达时准确率低 |
| ✅ 词性标注 | ❌ 不支持模糊匹配 |
| ✅ 社区最大（33K Stars） | - |

## 三、方案二：SnowNLP（文本分类）

### 3.1 基本信息

| 属性 | 说明 |
|------|------|
| **名称** | SnowNLP |
| **安装** | `pip install snownlp` |
| **大小** | ~5MB |
| **延迟** | 50-100ms |
| **Stars** | 6.6K+ (GitHub) |
| **特点** | 轻量级中文 NLP，内置文本分类（朴素贝叶斯） |

### 3.2 核心功能

| 功能 | 说明 |
|------|------|
| **中文分词** | Character-Based Generative Model |
| **词性标注** | TnT 3-gram 隐马 |
| **情感分析** | 返回 0-1 的积极概率 |
| **文本分类** | 朴素贝叶斯（**可自定义训练**） |
| **关键词提取** | TextRank 算法 |
| **文本摘要** | TextRank 算法 |
| **文本相似度** | BM25 算法 |
| **拼音转换** | Trie 树最大匹配 |
| **繁简转换** | Trie 树最大匹配 |

### 3.3 意图检测实现

```python
from snownlp import SnowNLP

class SnowNLPIntentDetector:
    """基于 SnowNLP 的意图检测器"""
    
    def __init__(self):
        # 关键词映射（与 Jieba 类似）
        self.intent_keywords = {
            "file": ["文件", "读取", "写入", "删除", "移动", "目录", "搜索文件"],
            "network": ["搜索", "下载", "http", "网址", "网络", "请求"],
            "system": ["进程", "CPU", "内存", "剪贴板", "系统"],
            "code": ["代码", "执行", "python", "运行"],
        }
    
    def detect(self, text: str) -> dict:
        """意图检测"""
        s = SnowNLP(text)
        
        # Step 1: 分词
        words = s.words
        
        # Step 2: 关键词提取（TextRank）
        keywords = s.keywords(5)
        
        # Step 3: 计算各意图得分
        scores = {}
        for intent, intent_kws in self.intent_keywords.items():
            score = 0
            for kw in intent_kws:
                if kw in text:
                    score += 2
            for word in words:
                if word in intent_kws:
                    score += 1
            for kw in keywords:
                if kw in intent_kws:
                    score += 1.5
            scores[intent] = score
        
        if not scores or max(scores.values()) == 0:
            return {"intent": "unknown", "confidence": 0, "sentiment": s.sentiments}
        
        best_intent = max(scores, key=scores.get)
        confidence = scores[best_intent] / (sum(scores.values()) + 0.01)
        
        return {
            "intent": best_intent,
            "confidence": round(confidence, 2),
            "sentiment": s.sentiments,  # 额外：情感分析
            "keywords": keywords,
        }

# 使用示例
detector = SnowNLPIntentDetector()
result = detector.detect("帮我读取 D:/test.txt 文件")
print(result)
```

### 3.4 SnowNLP 文本分类功能（关键）

SnowNLP 内置了**朴素贝叶斯文本分类器**，可以自定义训练：

```python
from snownlp import SnowNLP
from snownlp import sentiment

# 方法1：使用内置情感分析
s = SnowNLP("这个文件操作功能很好用")
print(s.sentiments)  # 0.97（积极）

# 方法2：自定义训练文本分类器
from snownlp.classification import BayesClassifier

# 准备训练数据
train_data = {
    "file": [
        "帮我读取这个文件",
        "把文件移动到新目录",
        "删除这个文件",
        "创建一个新文件夹",
        "搜索所有txt文件",
    ],
    "network": [
        "帮我搜索天气",
        "下载这个文件",
        "访问这个网址",
        "发送HTTP请求",
    ],
    "system": [
        "查看CPU使用率",
        "显示内存信息",
        "打开剪贴板",
        "关闭这个进程",
    ],
}

# 训练分类器
clf = BayesClassifier()
for intent, texts in train_data.items():
    for text in texts:
        clf.add(text, intent)
clf.train()

# 预测
result = clf.classify("帮我读取 D:/test.txt")
print(result)  # "file"
```

### 3.5 优缺点

| 优点 | 缺点 |
|------|------|
| ✅ 最小（~5MB） | ❌ 情感分析只对商品评价效果好 |
| ✅ 开箱即用 | ❌ 文本分类需要自定义训练数据 |
| ✅ 内置文本分类（朴素贝叶斯） | ❌ 社区较小（6.6K Stars） |
| ✅ 支持自定义训练 | ❌ 语义理解能力有限 |
| ✅ 功能丰富（情感/关键词/摘要/相似度） | ❌ 不支持模糊匹配 |
| ✅ 纯 Python 实现 | - |

## 四、方案三：sage-intent（关键词+LLM混合）

### 4.1 基本信息

| 属性 | 说明 |
|------|------|
| **名称** | sage-intent (isage-libs-intent) |
| **安装** | `pip install isage-intent` / `pip install isage-intent[llm]` |
| **大小** | ~5MB（基础）/ ~10MB（含 LLM） |
| **延迟** | <100ms（关键词）/ 200-500ms（LLM） |
| **特点** | 专门做意图识别，支持关键词+LLM混合 |

### 4.2 核心功能

| 功能 | 说明 |
|------|------|
| **IntentCatalog** | 意图目录管理，支持关键词和描述 |
| **KeywordIntentRecognizer** | 基于关键词的快速识别 |
| **LLMIntentRecognizer** | 基于 LLM 的语义识别 |
| **IntentClassifier** | 多识别器组合，支持投票/置信度/级联策略 |

### 4.3 意图检测实现

```python
from sage_intent import IntentCatalog, KeywordIntentRecognizer, IntentClassifier

# Step 1: 创建意图目录
catalog = IntentCatalog()
catalog.add_intent(
    name="file",
    keywords=["文件", "读取", "写入", "删除", "移动", "目录", "搜索", "复制", "粘贴"],
    description="文件操作：读取、写入、删除、移动、搜索文件和目录"
)
catalog.add_intent(
    name="network",
    keywords=["搜索", "下载", "http", "网址", "网络", "请求", "API"],
    description="网络操作：HTTP请求、Web搜索、API调用"
)
catalog.add_intent(
    name="system",
    keywords=["进程", "CPU", "内存", "剪贴板", "系统", "关机", "重启"],
    description="系统操作：进程管理、系统信息、剪贴板操作"
)

# Step 2: 创建关键词识别器
keyword_recognizer = KeywordIntentRecognizer(catalog)

# Step 3: 意图识别
intent = keyword_recognizer.recognize("帮我读取 D:/test.txt 文件")
print(intent.name)        # "file"
print(intent.confidence)  # 0.85
```

### 4.4 混合方案（关键词 + LLM）

```python
from sage_intent import IntentClassifier, KeywordIntentRecognizer, LLMIntentRecognizer

# 创建混合分类器
classifier = IntentClassifier(
    recognizers=[
        KeywordIntentRecognizer(catalog),     # 快速路径
        LLMIntentRecognizer(catalog, llm_client),  # 准确路径
    ],
    strategy="cascade"  # 级联：先关键词，失败再LLM
)

# 分类
intent = classifier.classify("帮我把这个文件的内容复制到剪贴板")
print(intent.name)  # "file"（关键词命中）或 "system"（LLM 判断）
```

### 4.5 三种策略

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| `vote` | 多个识别器投票 | 需要最高准确率 |
| `confidence` | 选择置信度最高的 | 需要快速决策 |
| `cascade` | 先快后慢级联 | **推荐**，兼顾速度和准确率 |

### 4.6 优缺点

| 优点 | 缺点 |
|------|------|
| ✅ **专门做意图识别** | ❌ 社区小（新项目，126 downloads/月） |
| ✅ 关键词+LLM混合方案 | ❌ LLM 模式需要额外 API 调用 |
| ✅ 三种组合策略 | ❌ 文档较少 |
| ✅ 架构清晰，可扩展 | ❌ 中文支持需要自定义关键词 |
| ✅ 级联模式兼顾速度和准确率 | - |

## 六、推荐方案

### 6.1 推荐矩阵

| 场景 | 推荐 | 原因 |
|------|------|------|
| **最轻量、最快** | Jieba | 10-50ms，15MB，分词+关键词提取 |
| **最简单集成** | SnowNLP | 5MB，开箱即用，功能丰富 |
| **专门做意图** | sage-intent | 内置意图识别，混合方案 |
| **50-70% 准确率** | Jieba 或 SnowNLP | 关键词匹配即可达到 |
| **70-85% 准确率** | sage-intent（混合模式） | 关键词+LLM |

### 6.2 最终推荐

**对于 OmniAgent 的当前需求（50-70% 准确率，轻量级）**：

| 优先级 | 推荐方案 | 理由 |
|--------|---------|------|
| **首选** | **Jieba** | 最轻量、最快、社区最大、功能够用 |
| 备选 | SnowNLP | 如果需要文本分类训练能力 |
| 未来 | sage-intent | 如果需要 LLM 混合方案 |
