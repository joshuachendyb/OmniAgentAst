### 9.2 完善ExecutionPanel数据结构

**需要修改的文件**: `frontend/src/components/Chat/ExecutionPanel.tsx`

**问题**:
ExecutionPanel期望的ExecutionStep类型与实际StreamStep类型不匹配

**ExecutionStep类型**（api.ts第445-451行）:
```typescript
export interface ExecutionStep {
  type: 'thought' | 'action' | 'observation' | 'error' | 'final';
  content?: string;
  tool?: string;
  params?: Record<string, any>;
  result?: any;
  timestamp: number;
}
```

**StreamStep类型**（api.ts第124-132行）:
```typescript
export interface StreamStep {
  type: 'thought' | 'action' | 'observation' | 'error' | 'interrupted' | 'final';
  step?: number;
  thought?: string;
  action?: string;
  action_input?: any;
  observation?: any;
  content?: string;
}
```

**修复方案**:
将StreamStep转换为ExecutionStep后再传递给ExecutionPanel

```typescript
// 在Chat组件中添加转换函数
const convertStreamToExecution = (step: StreamStep): ExecutionStep => {
  return {
    type: step.type,
    content: step.thought || step.content || step.action,
    tool: step.action,
    params: step.action_input,
    result: step.observation,
    timestamp: Date.now(),
  };
};

// 在onStep回调中使用转换
onStep: (step) => {
  const executionStep = convertStreamToExecution(step);
  setExecutionSteps(prev => [...prev, executionStep]);
}
```
